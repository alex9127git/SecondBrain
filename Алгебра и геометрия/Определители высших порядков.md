## Перестановки
**Перестановкой $n$ чисел $\lambda _1, \lambda _2, \dots, \lambda _n$** называется конечная последовательность $\alpha _1, \alpha _2, \dots, \alpha _n$, каждый член которой принадлежит множеству $\{\lambda _1, \lambda _2, \dots, \lambda _n\}$, при этом любые два элемента этой последовательности различны.

Пусть $\alpha _1, \alpha _2, \dots, \alpha _n$ — некоторая перестановка последовательности $1,2,\dots,n (n \in \mathbb{N})$. Пары элементов $\alpha _i, \alpha _j (i < j)$ образуют **инверсию (беспорядок)**, если $\alpha _i > \alpha _j$. Общее количество всех пар элементов перестановки, образующих инверсии, называют **числом инверсий** перестановки, или $\mathop{\text{inv}}(\alpha _1, \alpha _2, \dots, \alpha _n)$.

Для подсчёта числа инверсий в перестановке можно использовать следующую последовательность шагов:

1) Найти $k _1$ — количество элементов перестановки, стоящих перед числом $1$, затем вычеркнуть его.
2) Найти $k _2$ — количество элементов перестановки, стоящих перед числом $2$, затем вычеркнуть его.
3) Продолжаем выполнять эти шаги до $k _n$.
4) $\mathop{\text{inv}}(\alpha _1, \alpha _2, \dots, \alpha _n) = k _1 + k _2 + \dots + k _n$.

Пример:

$$
\mathop{\text{inv}}(4, 5, 1, 3, 2) = 2 + 3 + 2 + 0 + 0 = 7
$$

В зависимости от числа инверсий различают **чётные и нечётные** перестановки, то есть перестановки, содержащие чётное или нечётное количество инверсий соответственно.
