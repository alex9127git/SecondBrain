## Основные определения
Пусть Х и Y - некоторые [[Элементы теории множеств#Основные определения|множества]]. Декартово (прямое) произведение Х **×** Y - множество всех пар (х, у).
Х **×** Y = { (x, y) :- x ∈ Х, y ∈ Y}
### Пример
$$A = \{1; 2; 3\}\qquad
B = \{+; -\}$$

$$
B × A = \{(1; +), (1; -), (2; +), (2; -), (3; +), (3; -)\}
$$

Мощность декартова произведения $|A × B| = |A| * |B|$

Декартово произведение n множеств:
$X_{1} × X_{2} × X_{3} × ... × X_{n}$ =  $\{(x_{1} ; x_{2} ; x_{3} ; ... ; x_{n}) : x_{1} ∈ X_{1}, x_{2} ∈ X_{2}, ..., x_{n} ∈ X_{n}\}$ 

Декартово произведение вида $X × X × X × ... × X$ = $Х^n$ - декартова степень множества $Х$.

Свойства:
1) $(A \cup B) × C = (A × C) \cup (B × C)$
2) $(A \cap B) × C = (A × C) \cap (B × C)$
3) $(A \backslash B) × C = (A \cap \overline{B}) × C = (A × C) \cap (\overline{B} × C) = (A × C) \backslash (B × C)$

Доказательство 2 свойства:
1) Если $(x; y) ∈ (A \cap B) × C,$ тогда $(x; y) ∈ (A × C) \cap (B × C)$
2) Если $(x; y) ∈ (A × C) \cap (B × C)$, тогда $(х; y) ∈ (A \cap B) × C$

Пусть $(x; y) ∈ (A \cap B) × C$. Тогда$x ∈ (A \cap B), y ∈ C$ $⇒$ $x ∈ A, y ∈ C \text{ и } x ∈ B, y ∈ C$ $⇒$ $(x, y) ∈ A × C \text{ и } (x, y) ∈ B × C$. Отсюда $(x; y) ∈ (A × C) \cap (B × C)$

Пусть $(x; y) ∈ (A × C) \cap (B × C)$. Тогда $x ∈ A, y ∈ C \text{ и } x ∈ B, y ∈ C$ $⇒$ $y ∈ C, x ∈ A \cap B$ $⇒$ $(x, y) ∈ (A \cap B) × C$, ч.т.д.
## Отображение. Типы отображений.
Пусть $X$ и $Y$ - некоторые множества. **Отображением** $f : X → Y$ называется называется тройка $X, Y, f$, где $f$ - правило (закон), которое каждому $x$ из $X$ ставит в соответствие $y$ из $Y$.
## Пример
Отображение может быть задано формулой:

$$
Х = Y = R, y = x^3\qquad
$$

или напрямую:

$$
\begin{matrix}
Х = Y = \{1; 2; 3\}\\
f : X → Y\\
f(1) = 1\\
f(2) = 1\\
f(3) = 2
\end{matrix}
$$

Отображение $f : X → Y$ называется **сюръективным**, если

$$
∀ y ∈ Y, ∃ x ∈ X : f(x) = y
$$

Пояснение детальнее:
**Сюръективное отображение** это отображение между двумя множествами, при котором каждый элемент второго множества является образом хотя бы одного элемента первого множества. Это означает, что отображение "покрывает" все элементы второго множества своими значениями. Например, функция f(x)=2x+1 является сюръективным отображением множества действительных чисел на себя, так как для любого вещественного числа y существует такое  вещественное число x, такое что f(x)=yf(x)=y

Отображение f : X -> Y называется **инъективным**, если
$f(x_{1}) = f(x_{2}) ⇒ x_1 = x_2$
Это означает, что разные элементы из множества X имеют разные образы в множестве Y.
f(x) - образ, x - прообраз.
Иначе говоря, из равенства образов следует равенство прообразов.

Отображение f : X -> Y называется **биективным** (взаимно однозначным), если оно **сюръективно** и **инъективно**.
## Примеры
$$1.\ f(x) = x^2,  R^{+} = \{ x ∈ R : x >= 0 \}$$

$$f_{1}(x) = x^2 ; f_1 : R → R^+$$

$$∀ y ∈ R^+,  ∃ x ∈ R  : x = \pm\sqrt{y} ⇒ f_1(x) \text{ — сюръективно}$$
 
$$2.\ f_2(x) = x^2, f_2 : R^+ → R$$

$$∀ y ∈ R, ∃ x ∈ R^+ \text{ — неверно. Т.к. для y < 0 не существует f(x)}$$

$$x^2 = y < 0 ⇒ f_2 \text{ — не является сюръективным}$$

$$
\begin{matrix}
f_2(x_1) = f_2(x_2)\qquad{x_{1}}^2 =  {x_{2}}^2\qquad
\sqrt{{x_{1}}^2} =  \sqrt{{x_{2}}^2}\qquad
|{x_{1}}| =  |{x_{2}}|\\
x_1 = x_2 (x_1, x_2 ∈ R^+) ⇒ f_2 \text{ — инъективное}
\end{matrix}$$

$f_2$ = $x^2$, $f_2 : \ R^+ → R^+$ - биективное отображение
### Обратные отображения
Отображение $f : X → X$ называется **преобразованием множества Х**.

Тождественным преобразованием множества Х называется преобразование $e_x : e_{x}(x) = x, ∀ x ∈ X$
### Примеры
$$1.\ f : N → N, f(x) = 1 * x$$

$$
2.\ E_n = \begin{pmatrix}
    1 & 0 & \cdots & 0 \\
    0 & 1 & \cdots & 0 \\
    \vdots & \vdots & \ddots & \vdots \\
    0 & 0 & \cdots & 1
\end{pmatrix}
$$
$$f : X → X, f(X) = E * X$$

Пусть $f : X → Y, g : Y → Z$ — некоторые отображения. Тогда $gf : X → Z : (gf)(x) = g(f(x)), ∀ x ∈ X$ называется **суперпозицией** этих отображений.

$$f(x) = e^x, g(y) = \sin y ⇒ (gf)(x) = g(f(x)) = \sin e^x$$

Пусть $f : X → Y$ и $g : Y → X$. Тогда отображение $g$ называется обратным отображением к $f$ (отображение $f$ называется отображением к $g$), если выполнены 2 условия:

$$
\begin{cases}
    f(g(y)) = y \\
    g(f(x)) = x
\end{cases}
$$
## Теорема о существовании обратного отображения
Отображение $f : X → Y$ имеет обратное отображение тогда и только тогда, когда f является **биективным** отображением.

Доказательство необходимости:

Если $f$ имеет обратное отображение, то $f$ - биективное (инъективное И сюръективное). Пусть f имеет обратное отображение, то существует такое отображение $g : Y → X$, что 

$$
\begin{cases}
    f(g(y)) = y \\
    g(f(x)) = x
\end{cases}
$$

Покажем, что $f$ — сюръективное отображение: $∀ y ∈ Y, ∃x ∈ X : f(x) = y?$

Возьмём произвольный $y \in Y$ и найдём $x = g(y), x \in X$. Тогда $f(x) = f(g(y))$ и $f(x) = y$, из первого условия, отсюда $f$ — сюръективное отображение.

Покажем, что $f$ — инъективное отображение (из равенства образов следует равенство прообразов).

Пусть $f(x_1) = f(x_2), x_1, x_2  ∈ X$, тогда $x_1 = g(f(x_1)) = g(f(x_2)) = x_2 ⇒ x_2 = x_1$. Отсюда $f$ — инъективное отображение.

Тогда $f$ — биективное отображение, так как оно является одновременно инъективным и сюръективным.

Доказательство достаточности:

$f$ - биективное отображение. Докажем, что существует обратное отображение.

Из условия $f$ - инъективное и сюръективное отображение.

$∀ y ∈ Y, ∃ x ∈ X : f(x) = y$, так как $f$ сюръективно, — существует отображение $g : Y → X$, которое ставит в соответствие каждому $y \in Y$ свой $x \in X$.

Так как $f$ — инъективное, то $f(g(y_1)) = f(g(y_2)) ⇒ g(y_1) = g(y_2) ⇒ x_1 = x_2$. Отсюда:

$$
\begin{array}{c}
    \forall x \in X \\
    \forall y \in Y
\end{array}
\begin{cases}
    f(g(y)) = f(x) = y \\
    g(f(x)) = g(y) = x
\end{cases}
$$

$g$ — обратное отображение $f$, ч.т.д.
## Теорема о единственности обратного отображения
Если обратное отображение существует, то оно единственное.
## Пример построения обратного отображения
**Задача:** построить обратное отображение для $f : R → R$, $f(x) = \frac{2x - 5}{3}$, если это возможно.

Решение: Докажем, что $f$ — биективное отображение:

$$f(x_1) = f(x_2) => x_1 = x_2 (x_1, x_2 ∈ R)\qquad\frac{2x_1 - 5}{3} = \frac{2x_2 - 5}{3} => \ x_2 = x_1$$

Следовательно, $f$ — инъективно.

$$∀ y ∈ R, ∃ x ∈ R : f(x) = y\qquad y=\frac{2x-5}{3}\qquad3y = 2x-5\qquad3y+5=2x\qquad x=\frac{3y+5}{2}$$

Следовательно, $f$ — сюръективное отображение, для которого можно построить обратное отображение.
