## Теорема второго достаточно условия экстремума (№54)
**Пусть функция $y = f(x)$ [[Старшие производные#Определение|дважды]] [[Понятие производной#Определение|дифференцируема]] на некотором интервале $(c; d)$. вторая производная $f''(x)$ [[Непрерывные функции#Определения|непрерывна]] на этом интервале, первая производная в точке $a \in (c; d)$ равна нулю. Тогда, если вторая производная в точке $a$ меньше нуля (или равна нулю), то точка $a$ является точкой (нестрогого) локального максимума, если вторая производная в точке $a$ больше нуля (или равна нулю), то точка $a$ является точкой (нестрогого) локального минимума.**

Для доказательства используем разложение функции в ряд Тейлора. По условию теоремы функция дважды дифференцируема, поэтому строим разложение в окрестности точки $a$ до второй производной включительно:

$$
f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)(x-a)^2}{2!} + r_2(x)
$$

$$
f'(a) = 0 ⇒ f(x) - f(a) = \frac{f''(a)(x-a)^2}{2!} + r_2(x)
$$

$$
(x-a)^2 ⇒ f(x)-f(a) \text{ и } f''(a) \text{ имеют одинаковые знаки}
$$

При $f''(a) < 0$: $f(x) - f(a) < 0$, $f(x) < f(a)$ в окрестности точки $a$, то есть $a$ — локальный максимум.

При $f''(a) > 0$: $f(x) - f(a) > 0$, $f(x) > f(a)$ в окрестности точки $a$, то есть $a$ — локальный минимум.
## Выпуклые функции
Функция $y = f(x)$ называется выпуклой вниз, если её график лежит ниже любой хорды, но выше любой касательной.

Функция $y = f(x)$ называется выпуклой вверх, если её график лежит выше любой хорды, но ниже любой касательной.

Покажем это, проведя хорду через параболу.

Пусть $\xi \in (x, y)$. Тогда мы можем выразить $\xi$ через $x$ и $y$:

$$
\xi = \alpha x + (1 - \alpha)y
$$

Таким образом, при $\alpha$ = 0: $\xi = y$, а при $\alpha$ = 1: $\xi = x$.

Тогда ордината функции в точке $\xi$ будет равно $f(\alpha x + (1-\alpha)y)$, а ордината хорды в точке $\xi$ будет равна $\alpha f(x) + (1 - \alpha) f(y)$.

Тогда для выпуклой вниз функции будет справедливо

$$
f(\alpha x + (1-\alpha)y) < \alpha f(x) + (1 - \alpha) f(y),
$$

а для выпуклой вверх будет справедливо

$$
f(\alpha x + (1-\alpha)y) > \alpha f(x) + (1 - \alpha) f(y).
$$

## Теорема о первом достаточном условии выпуклости функции (№55)
**Пусть функция $y = f(x)$ дифференцируема на интервале $(a; b)$ и непрерывна на отрезке $[a; b]$. Тогда, если первая производная монотонно возрастает (убывает) на интервале $(a; b)$, то функция будет выпукла вниз (вверх).**

Доказательство:

Сделаем дополнительное построение, взяв точки $\xi_1$ и $\xi_2$ так, что $x < \xi_1 < \xi < \xi_2 < y$.

Проверим условие выпуклости ($\alpha$ и $\beta$ — коэффициенты, дающие в сумме 1):

$$
f(\alpha x + \beta y) - \alpha f(x) - \beta f(y) = (\alpha+\beta)f(\alpha x + \beta y) - \alpha f(x) - \beta f(y) = 
$$

$$
\alpha f(\alpha x + \beta y) - \alpha f(x) + \beta f(\alpha x + \beta y) - \beta f(y) = \alpha(f(\alpha x + \beta y) - f(x)) + \beta(f(\alpha x + \beta y) - f(y))
$$

Применяем формулу Лагранжа для конечных приращений в частном случае к каждой скобке:

$$
\alpha(f(\alpha x + \beta y) - f(x)) + \beta(f(\alpha x + \beta y) - f(y)) = \alpha f'(\xi_1)(\alpha x + \beta y - x) - \beta f'(\xi_2)(y - \alpha x - \beta y) = 
$$

$$
= \alpha f'(\xi_1)(x(\alpha - 1) + \beta y) - \beta f'(\xi_2)(\alpha x - y(1 - \beta)) = \alpha f'(\xi_1)(x(\alpha - 1) + \beta y) - \beta f'(\xi_2)(y(1 - \beta) + \alpha x) =
$$

$$
= \alpha f'(\xi_1)(\beta y - \beta x) - \beta f'(\xi_2)(\alpha y - \alpha x) = \alpha \beta f'(\xi_1)(y-x) - \alpha \beta f'(\xi_2)(y-x)=
$$

$$
=\alpha\beta(y-x)(f'(\xi_1) - f'(\xi_2))
$$

$\alpha > 0$, $\beta > 0$, $y - x > 0 ⇒$ знак выражения $f(\alpha x + \beta y) - \alpha f(x) - \beta f(y)$ совпадает со знаком выражения $f'(\xi_1) - f'(\xi_2)$.

$$
f'(\xi_2) \le f'(\xi_1) ⇒ f'(\xi_1) - f'(\xi_2) \ge 0 ⇒ f(\alpha x + \beta y) - \alpha f(x) - \beta f(y) \ge 0 ⇒
$$

$$
⇒ f(\alpha x + \beta y) \ge \alpha f(x) + \beta f(y)
$$

Производная монотонно убывает, функция выпукла вверх.

$$
f'(\xi_2) \ge f'(\xi_1) ⇒ f'(\xi_1) - f'(\xi_2) \le 0 ⇒ f(\alpha x + \beta y) - \alpha f(x) - \beta f(y) \le 0 ⇒
$$

$$
⇒ f(\alpha x + \beta y) \le \alpha f(x) + \beta f(y)
$$

Производная монотонно возрастает, функция выпукла вниз, ч.т.д.
## Теорема о втором достаточном условии выпуклости функции (№56)
**Пусть функция $y = f(x)$ дважды дифференцируема на некотором числовом промежутке и непрерывна на нём. Тогда для выпуклости функции необходимо и достаточно, чтобы вторая производная была знакопостоянна.**

Доказательство необходимости:

Рассмотрим промежуточную точку $a$ из данной области определения. В качестве $x$ возьмём сначала точку $x = a + h$, где $h$ — малый шаг дискретизации. Используем разложение функции в ряд Тейлора:

$$
f(a+h) = f(a) + f'(a)h + \frac{f''(a)h^2}{2} + o(h^2)
$$

Теперь возьмём $x = a - h$ и снова раскладываем в ряд Тейлора:

$$
f(a - h) = f(a) - f'(a)h + \frac{f''(a)h^2}{2} + o(h^2)
$$

Разделим оба уравнения на 2 и сложим их:

$$
\frac{f(a+h)}{2} = \frac{f(a)}{2} + \frac{f'(a)h}{2} + \frac{f''(a)h^2}{4} + o(h^2)
$$

$$
\frac{f(a - h)}{2} = \frac{f(a)}{2} - \frac{f'(a)h}{2} + \frac{f''(a)h^2}{4} + o(h^2)
$$

$$
\frac{f(a+h)+f(a-h)}{2} = f(a) + \frac{f''(a)h^2}{2}+o(h^2)
$$

$$
\frac{f(a+h)+f(a-h)}{2} - f(a) = \frac{f''(a)h^2}{2} + o(h^2)
$$

Перепишем левую часть высказывания:

$$
\frac{f(a+h)+f(a-h)}{2} - f(a) = \frac{1}{2}f(a-h)+\frac{1}{2}f(a+h)-f(\frac{1}{2}(a-h)+\frac{1}{2}(a+h))
$$

Пусть $x = a - h$ и $y = a + h$, $\alpha = \beta = \frac{1}{2}$. Тогда:

$$
\frac{1}{2}f(a-h)+\frac{1}{2}f(a+h)-f(\frac{1}{2}(a-h)+\frac{1}{2}(a+h)) = \alpha f(x) + \beta f(y) - f(\alpha x + \beta y)
$$

Получается, что

$$
\alpha f(x) + \beta f(y) - f(\alpha x + \beta y) = \frac{f''(a)h^2}{2} + o(h^2)
$$

Знаки этих выражений совпадает, значит, при выпуклости функции знак второй производной сохраняется, ч.т.д.

Доказательство достаточности:

Из выражения:

$$
\alpha f(x) + \beta f(y) - f(\alpha x + \beta y) = \frac{f''(a)h^2}{2} + o(h^2)
$$

следует, что знаки выражений одинаковые. При сохранении знака второй производной сохраняется и выпуклость функции, ч.т.д.
## Точки перегиба
Точка $x_0$ называется точкой перегиба функции, если по одну сторону от этой точки функция выпукла вверх, а по другую сторону — выпукла вниз.
## Теорема о необходимом и достаточном условия существования точки перегиба (№57)
**Пусть функция $y = f(x)$ трижды дифференцируема на некотором промежутке, третья производная непрерывна на нём, $f''(x _0) = 0, f'''(x _0) \ne 0$. Тогда точка $x _0$ является точкой перегиба.**

Доказательство:

По условию теоремы функция трижды дифференцируема, разложим функцию в ряд Тейлора в окрестности точки $a$ до третьей производной включительно:

$$
f(x)=f(a) + f'(a)(x-a) + \frac{f''(a)(x-a)^2}{2!} + \frac{f'''(a)(x-a)^3}{3!} + r_3(x)
$$

$$
f''(a) \ne 0 ⇒ f(x) - f(a) = f'(a)(x-a) + \frac{f'''(a)(x-a)^3}{3!} + r_3(x)
$$
