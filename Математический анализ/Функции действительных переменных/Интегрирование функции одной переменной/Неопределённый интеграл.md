## Определение первообразной
В теории дифференциального исчисления мы находили [[Понятие производной#Определение|производную]] и [[Дифференциал функции#Определение|дифференциал]] некоторой функции $F(x)$ следующим образом:

$$
F'(x)=\lim_{\Delta x \to 0} \frac{F(x+\Delta x) - F(x)}{\Delta x}\qquad \mathrm{d}F(x) = F'(x)\ \mathrm{d}x
$$

Введём обозначение: пусть $F'(x) = f(x)$. Тогда

$$
\mathrm{d}F(x) = f(x)\ \mathrm{d}x
$$

В интегральном исчислении решается обратная задача: отыскание функции $F(x)$ по её заданной производной $f(x)$ или дифференциалу $f(x)\ \mathrm{d}x$.

Функция $F(x)$ называется первообразной функцией для функции $f(x)$ на множестве $X$, если

$$
F'(x) = f(x)\qquad \mathrm{d}F(x) = f(x)\ \mathrm{d}x
$$
## Основная теорема интегрального исчисления (№58)
**Пусть $F _1(x)$, $F _2(x)$ — любые две первообразные для функции $f(x)$ на промежутке $X$. Тогда:**

$$
\forall x \in X : F_2(x) - F_1(x) = \text{const}
$$

Доказательство:

Обозначим через $F(x)$ разность $F _2(x) - F _1(x)$:

$$
F'(x) = F'_2(x) - F'_1(x) = f(x) - f(x) = 0
$$

$$
F'(x)=0 ⇒ F(x) = \text{const}
$$

, ч.т.д.

Следствие: Пусть $F(x)$ — одна из первообразных для функции $f(x)$. Тогда любая первообразная $\Phi(x)$ может быть записана в виде $F(x) + C, C = \text{const}$.

Совокупность всех первообразных на промежутке $X$ называется неопределённым интегралом для функции $f(x)$ и обозначается как $\int f(x)\ \mathrm{d}x$. В этом обозначении $f(x)$ называется подынтегральной функцией, а $f(x)\ \mathrm{d}x$ — подынтегральным выражением.

$$
\int f(x)\ \mathrm{d}x = F(x)+C, C = \text{const}
$$

Пример:

$$
\int \cos x \ \mathrm{d}x = \sin x + C
$$

Замечание: название переменной после $\mathrm{d}$ указывает, по какой переменной ведётся интегрирование.

Пример:

$$
\int xy\ \mathrm{d}x = \frac{x^2}{2}y + C
$$
## Таблица основных интегралов

1. $\int x^n\ \mathrm{d}x = \frac{x^{n+1}}{n+1}+C$
2. $\int \frac{\mathrm{d}x}{x} = \ln |x| + C$
3. $\int \frac{\mathrm{d}x}{1+x^2} = \mathop{\text{arctg}} x + C$
4. $\int \frac{\mathrm{d}x}{1-x^2} = \frac{1}{2} \ln |\frac{1+x}{1-x}| + C$
5. $\int \frac{\mathrm{d}x}{\sqrt{1-x^2}} = \arcsin x + C$
6. $\int \frac{\mathrm{d}x}{\sqrt{x^2 \pm 1}} = \ln |x + \sqrt{x^2 \pm 1}| + C$
7. $\int a^x\ \mathrm{d}x = \frac{a^x}{\ln a} + C$
8. $\int e^x\ \mathrm{d}x = e^x + C$
9. $\int \sin x\ \mathrm{d}x = -\cos x + C$
10. $\int \cos x\ \mathrm{d}x = \sin x + C$
11. $\int \frac{\mathrm{d}x}{\sin^2 x} = -\mathop{\text{ctg}} x + C$
12. $\int \frac{\mathrm{d}x}{\cos^2 x} = \mathop{\text{tg}} x + C$
## Основные свойства неопределённых интегралов
1. $\int \mathrm{d}F(x) = F(x) + C$
2. $\mathrm{d}\int f(x)\ \mathrm{d}x = f(x)\ \mathrm{d}x$
3. $\int (f(x) \pm g(x))\ \mathrm{d}x = \int f(x) \mathrm{d}x + \int g(x)\ \mathrm{d}x$
4. $\int k f(x)\ \mathrm{d}x = k\int f(x)\ \mathrm{d}x, k = \text{const}$

Первые два свойства доказываются, исходя из определения неопределённого интеграла.

Докажем третье свойство:

Пусть $F(x)$ — первообразная для $f(x)$, $G(x)$ — первообразная для $g(x)$. Тогда 

$$
\int f(x)\ \mathrm{d}x = F(x) + C_1\qquad \int g(x)\ \mathrm{d}x = G(x) + C_2
$$

Сложим эти два равенства:

$$
\int f(x)\ \mathrm{d}x + \int g(x)\ \mathrm{d}x = F(x) + G(x) + C, C = C_1 + C_2
$$

Покажем, что функция $F(x) + G(x) + C$ является первообразной для подынтегральной функции $f(x) + g(x)$ в выражении слева:

$$
(F(x)+G(x))' = F'(x) + G'(x) = f(x) + g(x)
$$

Для разности доказывается аналогично.
