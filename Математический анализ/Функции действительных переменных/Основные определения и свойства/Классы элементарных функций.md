## Основные классы элементарных функций
1) $y = x^a$ — степенные
2) $y = a^x$ — показательные
3) $y = log_{a} x$ — логарифмические
4) $y = \sin x, y = \cos x, y = \tan x, y = \cot x$ — тригонометрические
5) $y = \arcsin x, y = \arccos x, y = \arctan x, y = arcctg(x)$ — обратные тригонометрические
## Свойства степенных функций
Функция $y = x^n, n ∈ N, x ∈ [0; +∞)$ непрерывна на данной области определения.

При $n = 1$ $f(x _n) = y _n = x _n, f(x _0) = y _0 = x _0 ⇒ \{f(x _n)\} \xrightarrow{n→∞}x _0, x _0 = f(x _0)$ — функция непрерывна.

При $n > 1$ $y = x^n$ непрерывна как [[Непрерывные функции#Арифметические операции над непрерывными функциями (№31)|произведение n непрерывных функций]].

Функция $y = x^n, n ∈ N, x ∈ [0; +∞)$ возрастает на данной области определения.

$$
\forall x _1, x _2 ∈ [0; +∞) : x _2 > x _1 ⇒ {x _2}^n \ge {x _1}^n
$$

$$
({x _2}^n - {x _1}^n) = (x _2 - x _1)({x _1}^{n-1} + {x _1}^{n-2}x _2 + {x _1}^{n-3}{x _2}^2 + \dots + {x _2}^{n-1})
$$

Оба множителя в правой части выражения положительные, поэтому ${x _2}^n - {x _1}^n > 0$ для всех $x ∈ [0; +∞)$.

При $x ∈ (-∞; 0]$ поведение функции зависит от значения $n$. При чётных значениях $n$ функция убывает в этой области, при нечётных — возрастает.

$$
\forall x _1, x _2 ∈ [0; +∞) : x _2 > x _1 ⇒ {x _2}^{2n} \le {x _1}^{2n}
$$

$$
({x _2}^{2n} - {x _1}^{2n}) = (x _2 - x _1)({x _1}^{2n-1} + {x _1}^{2n-2}x _2 + {x _1}^{2n-3}{x _2}^2 + \dots + {x _2}^{2n-1}) < 0
$$

(отрицательное число в нечётной степени является отрицательным, т.е. второй множитель является отрицательным)

или

$$
\forall x _1, x _2 ∈ [0; +∞) : x _2 > x _1 ⇒ {x _2}^{2n+1} \ge {x _1}^{2n+1}
$$

$$
({x _2}^{2n+1} - {x _1}^{2n+1}) = (x _2 - x _1)({x _1}^{2n} + {x _1}^{2n-1}x _2 + {x _1}^{2n-2}{x _2}^2 + \dots + {x _2}^{2n}) > 0
$$

(отрицательное число в чётной степени является положительным, т.е. второй множитель является положительным)

Рассмотрим функцию $y = x^p, p = \frac{m}{n}; m, n ∈ Z$.

Функция $y = x^{\frac{1}{n}}$ непрерывна как обратная функции $y = x^n$. Соответственно, функция $y = x^{\frac{m}{n}}$ непрерывна как произведение нескольких непрерывных функций. Монотонность доказывается аналогично.

График этой функции доступен по ссылке: https://www.desmos.com/calculator/ftpd6j0inw?lang=ru
## Свойства показательных функций
Функция $y = a^x$ непрерывна на всей числовой оси.

При $0 < a < 1$ функция убывает на промежутке $x ∈ (-∞; +∞)$; при $a > 1$ функция возрастает на промежутке $x ∈ (-∞; +∞)$.

График этой функции доступен по ссылке: https://www.desmos.com/calculator/za31shx29s?lang=ru
## Свойства логарифмических функций
Функция $y = log_{a} x$ является обратной для показательной функции $y = a^x$; по [[Обратные функции#Теорема об обратной функции (№36)|теореме об обратной функции]] наследует все её свойства.

При $0 < a < 1$ функция убывает на промежутке $x ∈ (-∞; +∞)$; при $a > 1$ функция возрастает на промежутке $x ∈ (-∞; +∞)$.

## Тригонометрические функции
Функции $y = \sin x$ и $y = \cos x$ являются непрерывными на всей числовой оси.

Докажем, что $\lim _{x→x _0} \sin x = x _0$.

$$
|\sin x-\sin x _0| = |2 \sin(\frac{x - x _0}{2}) \cos (\frac{x + x _0}{2})| = 2|\sin(\frac{x - x _0}{2})||\cos (\frac{x + x _0}{2})|
$$

Оба модуля всегда меньше или равны единицы, поэтому

$$
2|\sin(\frac{x - x _0}{2})||\cos (\frac{x + x _0}{2})| \le 2|\sin(\frac{x - x _0}{2})|
$$

$$
|\sin(x)| \le |x| ⇒ 2|\sin(\frac{x - x _0}{2})| \le 2 \frac{|x - x _0|}{2} = |x - x_0| ⇒ |\sin(x) - \sin(x _0)| \le |x - x _0|
$$

Запишем определение по Коши для того предела, который мы хотим доказать:

$$
\forall ε > 0 : \exists \delta = \delta(\varepsilon) : |x-x _0| < \delta, |\sin(x)-\sin(x _0)| < \varepsilon
$$

Получается, что если разность $|x - x _0|$ достаточно малая, то и $|\sin(x)-\sin(x _0)|$ тоже будет  достаточно мала, что доказывает непрерывность функции.

Непрерывность функции $y = \cos x$ доказывается аналогично.
