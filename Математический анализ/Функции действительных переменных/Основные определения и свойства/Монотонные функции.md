## Определение
Пусть [[Понятие функции#Определение|функция]] $y = f(x)$ определена на [[Понятие множества#Определение|множестве]] $D$. Функция называется монотонно возрастающей на множестве $D$, если $\forall x _1, x _2 ∈ D : x _1 < x _2 ⇒ f(x _1) \le f(x _2)$, Если знак строгий ($f(x _1) < f(x _2)$), тогда функция строго монотонно возрастает.

Функция называется монотонно убывающей на множестве $D$, если $\forall x _1, x _2 ∈ D : x _1 < x _2 ⇒ f(x _1) \ge f(x _2)$, Если знак строгий ($f(x _1) > f(x _2)$), тогда функция строго монотонно убывает.

Примеры:
1. $y = x^2$. монотонно возрастает на $D ∈ [0; +∞)$; монотонно убывает на $D ∈ (-∞; 0]$.
2. $y = c = const$. монотонно возрастает и убывает на $D ∈ (-∞; +∞)$.
3. $y = \sin x$. монотонно возрастает на $D ∈ [-\frac{\pi}{2} + 2\pi n; \frac{\pi}{2} + 2\pi n], n ∈ \mathbb{Z}$; монотонно убывает на $D ∈ [\frac{\pi}{2} + 2\pi n; \frac{3\pi}{2} + 2\pi n], n ∈ \mathbb{Z}$.

## Критерий непрерывности для монотонной функции (№34)
**Пусть функция $y = f(x)$ монотонно возрастает или монотонно убывает на промежутке $[a; b]$. На концах отрезка функция принимает значения $f(a) = \alpha$, $f(b) = \beta$. Для того чтобы эта функция являлась непрерывной, необходимо и достаточно, чтобы её значения полностью заполняли промежуток $[\alpha; \beta]$.**

Для определённости будем полагать, что функция $y = f(x)$ монотонно возрастает.

Доказательство необходимого условия: Пусть функция непрерывна. Тогда в силу [[Непрерывные функции#Следствие из теоремы Коши о промежуточном значении непрерывной функции (№32)|следствия из теоремы Коши]] её значения полностью заполняют промежуток $[\alpha; \beta]$.

Доказательство достаточного условия: Возьмём произвольное достаточно малое значение $\varepsilon$. В силу того, что значения функции полностью заполняют промежуток $[\alpha; \beta]$:

$$
\exists x _1 : f(x _1) = f(x _0) - \varepsilon; \exists x _2 : f(x _2) = f(x _0) + \varepsilon
$$

Возьмём $x$ такой, что $x _1 < x < x _2$. В силу монотонности функции $f(x _1) < f(x) < f(x _2) ⇒ f(x _0) - \varepsilon < f(x) < f(x _0) + \varepsilon ⇒$ $|f(x) - f(x _0)| < \varepsilon ⇒ |x - x _0| < \delta, \delta = \delta(\varepsilon)$.

Получили [[Предел функции#Определение по Коши (на языке ε — δ)|определение предела функции]] на языке $\varepsilon-\delta$:

$$
\forall ε > 0 : \exists \delta = \delta(\varepsilon) : |x-x _0| < \delta, |f(x)-f(x _0)| < \varepsilon
$$

Выполняется [[Непрерывные функции#Определения|определение непрерывности функции в точке]]: $\lim _{x→x _0} f(x) = f(x _0)$. Поскольку точки $x$ и $x _0$ была выбрана нами произвольно, это будет справедливо для любой точки отрезка $[a; b]$.
