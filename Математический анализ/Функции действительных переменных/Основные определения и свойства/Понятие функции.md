## Определение
Функция — правило, закон, отображение, по которому каждому значению $x$ из $D$, где $D$ — область существования функций, ставят в соответствие значение $y$ из $E$, где $E$ — область значения функции.

Способы задания функций:

1. аналитический
2. графический
3. табличный (функция имеет дискретный вид)
4. с помощью логических выражений

Примеры функций, заданных логическими выражениями
1. функция знака:

$$
\mathop{\text{sign}}(x)=
\begin{cases}
1, x > 0\\
0, x = 0\\
-1, x < 0
\end{cases}
$$

2. функция Дирихле:

$$
D(x) = 
\begin{cases}
1, x ∈ \mathbb{Q}\\
0, x ∈ \mathbb{I}
\end{cases}
$$

где $\mathbb{Q}$ — множество рациональных чисел, $\mathbb{I}$ — множество иррациональных чисел.

3. функция Римана:

$$
R(x) = 
\begin{cases}
0, x ∈ \mathbb{I}\\
\frac{1}{q}, x = \frac{p}{q}\quad\text{(то есть $x$ — рациональное число)}
\end{cases}
$$

## Точки сгущения
Точка $a$ [[Понятие множества#Знаки принадлежности|∈]] $D$ называется точкой сгущения, если в её любой сколько угодно малой $\varepsilon$-окрестности находится бесконечно много точек этого множества.
