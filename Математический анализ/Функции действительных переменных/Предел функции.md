## Определение по Гейне (на языке последовательности)
Пусть $a$ — [[Понятие функции#Точки сгущения|точка сгущения]]. Говорят, что $\lim_{x→a} f(x) = b$, если

$$
\forall \{x _n\} \xrightarrow{n→∞}a ⇒ \{f(x _n)\} \xrightarrow{n→∞} b
$$

Данное определение описывает предел функции с помощью определения [[Пределы числовых последовательностей#Определение|предела для числовой последовательности]].
## Определение по Коши (на языке ε — δ)
Пусть $a$ — точка сгущения. Говорят, что $\lim_{x→a} f(x) = b$, если

$$
\forall ε > 0 : \exists \delta = \delta(\varepsilon) : |x-a| < \delta, |f(x)-b| < \varepsilon
$$

При сужении $\delta$-окрестности сузится и $\varepsilon$-окрестность; значение $f(x)$ будет всё ближе к $b$.

Геометрическая интерпретация этих двух определений доступна по ссылке: https://www.desmos.com/calculator/lohcceadvj?lang=ru
## Теорема №29
**Определения по Коши и по Гейне идентичны. Если функция имеет предел в смысле одного определения, то она имеет предел в смысле другого.**

Доказательство:

Пусть верно определение по Коши. Докажем, что определение по Гейне тоже верно.

$$
\forall ε > 0 : \exists \delta = \delta(\varepsilon) : |x-a| < \delta, |f(x)-b| < \varepsilon
$$

Рассмотрим последовательность $\forall \{x _n\} \xrightarrow{n→∞}a$. Подставим $x _n$ вместо $x$:

$$
|x_n - a| < \delta, |f(x_n) - b| < \varepsilon ⇒ \lim_{n→∞} f(x_n) = b,
$$

выполняется определение предела по Гейне.

Пусть верно определение по Гейне. Докажем, что определение по Коши тоже верно.

Предположим противное:

$$
\exists \varepsilon_0 > 0 : \forall \delta = \delta(\varepsilon_0) : |x-a|<\delta, |f(x)-b|>\varepsilon_0
$$

Подставим последовательность $x_n$ в первое неравенство:

$$
|x_n - a| < \delta ⇒ |f(x_n) - b| > \varepsilon_0
$$

Получили противоречие с определением предела по Гейне:

$$
|f(x_n) - b| < \varepsilon ⇒ \lim_{n→∞} f(x_n) = b,
$$

ч.т.д.
## Односторонние пределы
$$
\lim _{x→a+0} f(x) = b \text{ — предел справа}
$$

$$
\text{По Гейне: }\forall \{x _n\}>a, \{x _n\}\xrightarrow{n→∞}a, \{f(x _n)\}\xrightarrow{n→∞}b
$$

$$
\text{По Коши: }\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : 0 < x - a < b, |f(x) - b| < \varepsilon
$$

$$
\text{Пример: }\lim _{x→0-0} sign(x)=-1
$$

$$
\lim _{x→a-0} f(x) = b \text{ — предел слева}
$$

$$
\text{По Гейне: }\forall \{x _n\}<a, \{x _n\}\xrightarrow{n→∞}a, \{f(x _n)\}\xrightarrow{n→∞}b
$$

$$
\text{По Коши: }\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : 0 < a - x < b, |f(x) - b| < \varepsilon
$$

$$
\text{Пример: }\lim _{x→0+0} sign(x)=1
$$
