## Билет 1: Числовые множества. Операции над множествами
### Определение множества
Понятие множества в математическом анализе строго неопределимо. 

Под множеством понимают набор, совокупность, собрание некоторых объектов, которые называются элементами множества.

Множества обычно обозначаются заглавными латинскими буквами (A, B, C...), а их элементы - строчными латинскими буквами (a, b, c...).

Вводится понятие пустого (∅) множества — множества, которое не содержит элементов.
### Операции принадлежности
Запись $a ∈ A$ означает, что элемент принадлежит множеству А.

$a ∉ A$ означает, что элемент a не принадлежит множеству A.

$A ⊂ B$ означает, что A является подмножеством множества B (все элементы множества A принадлежат множеству B).

$$A ⊂ B ⇔ B ⊃ A$$

Запись A ⊂ B можно также расписать подлиннее, используя кванторы:

$$∀ a ∈ A ⇒ a ∈ B$$
#### Примеры
1. $A = \{1, 2, 3, 4, 5\}$
2. $B = \{x : x² = -1\} = ∅$
3. $C = \text{\{"множество книг в библиотеке ВГУ"\}}$
### Операция объединения
⋃ — логическое ИЛИ, логическое сложение, объединение
> A ⋃ B = {x :- x [[1 семестр БИЛЕТЫ 1-28#Операции принадлежности|∈]] A или  x ∈ B}
#### Примеры
1. A = {1, 2, 3}; B = {4, 5}; A ⋃ B = {1, 2, 3, 4, 5}
2. A = {множество квартир 1-го подъезда}
   B = {множество квартир 2-го подъезда}
   A ⋃ B = {множество всех квартир двухподъездного дома}

### Операция пересечения
⋂ — логическое И, логическое произведение, пересечение
> A ⋂ B = {x :- x ∈ A и x ∈ B}
#### Примеры
1. A = {1, 2, 3}; B = {2, 3}; A ⋂ B = {2, 3}
2. A = {множество квартир, где есть холодная вода}
   B = {множество квартир, где проведено электричество}
   A ⋂ B = {множество квартир, где есть и холодная вода, и электричество}
## Билет 2: Теоремы о точных гранях множеств
### Определение точных граней
[[1 семестр БИЛЕТЫ 1-28#Определение множества|Множество]] $E$ [[1 семестр БИЛЕТЫ 1-28#Операции принадлежности|⊂]] $\mathbb{R}$ называется ограниченным сверху, если

$$∃ a ∈ \mathbb{R} : ∀ x ∈ E: x ≤ a$$

Перевод на русский: существует вещественное число a такое, что для каждого элемента x множества E выполняется условие: x меньше или равно a.

Число a называют верхней гранью множества. Так, для множества отрицательных чисел E = (-∞; 0) любое число из отрезка \[0; +∞\) является его верхней гранью.

Наименьшая из всех верхних граней множества называется его точной верхней гранью, или супремумом. (a = sup E)

Множество $E ⊂ \mathbb{R}$ называется ограниченным снизу, если

$$∃ a ∈ \mathbb{R} : ∀ x ∈ E: x ≥ a$$

т.е. существует вещественное число a такое, что для каждого элемента x множества E выполняется условие: x больше или равно a.

В этом случае число a называется нижней гранью множества.

Наибольшая из всех нижних граней множества называется его точной нижней гранью или инфимумом. (a = inf E)

Множество $E ⊂ \mathbb{R}$ называется ограниченным, если оно ограничено и сверху и снизу, т.е.

$$∃ a, b ∈ \mathbb{R} : ∀ x ∈ E: a ≤ x ≤ b$$
### Запись определений точных граней с помощью формул
a = sup E. если выполняются два условия:
1. $∀ x ∈ E: x ≤ a$
2. $∀ ε > 0$ ($ε$ - число, по смыслу являющееся очень маленьким): $∃ x ∈ E : a - ε < x < a$
Первое условие показывает, что $a$ является верхней гранью множества $E$, и все числа этого множества меньше верхней грани. Второе условие показывает, что $a$ является **точной** верхней гранью множества $E$, и хотя бы одно число находится очень близко к этой верхней грани.

a = inf E. если выполняются два условия:
1. $∀ x ∈ E: x ≥ a$
2. $∀ ε > 0$ ($ε$ - число, по смыслу являющееся очень маленьким): $∃ x ∈ E : a < x < a + ε$
Первое условие показывает, что $a$ является нижней гранью множества $E$, и все числа этого множества больше верхней грани. Второе условие показывает, что $a$ является **точной** нижней гранью множества $E$, и хотя бы одно число находится очень близко к этой нижней грани.
### Теорема
**Всякое непустое ограниченное сверху (снизу) множество имеет точную верхнюю (нижнюю) грань.**

Доказательство: Пусть множество E ограничено сверху. Для ограниченного снизу множества доказательство аналогично.

Точную верхнюю грань будем искать в самом множестве по следующему алгоритму:
1. На первом этапе отберем те числа, у которых наибольшая целая часть. Выделим её и обозначим как $x_0$.
2. Из этих чисел выберем те, у которых наибольший первый знак после запятой, и обозначим его через $x_1$.
3. Продолжаем выполнять шаг 2, пока из выбранных чисел не останется одно число. Оно и будет являться точной верхней гранью.
## Билет 3: Числовые последовательности. Бесконечно малая последовательность. Ограниченность БМП
### Определение числовой последовательности
Числовой последовательностью $\{a_n\}^{∞} _{n=1}$ называют бесконечный упорядоченный набор чисел, расположенный в порядке возрастания их номеров.

Пример: $a_n = \frac{1}{n}$, n [[1 семестр БИЛЕТЫ 1-28#Операции принадлежности|∈]] $\mathbb{N}$ — последовательность, заданная формулой её общего члена.

$$a_1 = 1, a_2 = \frac{1}{2}, a₃ = \frac{1}{3}, \dots, aₙ = \frac{1}{n}$$

Замечание: [[1 семестр БИЛЕТЫ 1-28#Определение множества|понятие множества]] является более широким понятием, нежели последовательность, т.к. элементы множества не обязаны быть пронумерованы.
### Ограниченные последовательности
Числовая последовательность $\{a_n\}^{∞} _{n=1}$ называется ограниченной, если все её члены не превосходят по модулю некоторое значение $m$:

$$∃ m ∈ \mathbb{R} : ∀ n ∈ \mathbb{N}: |a_n| ≤ m$$

...ограниченной сверху, если все её члены меньше некоторого значения $m$:

$$∃ m ∈ \mathbb{R} : ∀ n ∈ \mathbb{N}: a_n ≤ m$$

...ограниченной снизу, если все её члены больше некоторого значения $m$:

$$∃ m ∈ \mathbb{R} : ∀ n ∈ \mathbb{N}: a_n ≥ m$$

x = sup $\{a_n\}^{∞} _{n=1}$, если
1. $∀ n ∈ \mathbb{N}: a_n ≤ x$
2. $∀ ε > 0 : ∃ n ∈ \mathbb{N} : x - ε < a_n < x$

x = inf $\{a_n\}^{∞} _{n=1}$, если
1. $∀ n ∈ \mathbb{N}: a_n ≥ x$
2. $∀ ε > 0 : ∃ n ∈ \mathbb{N} : x < a_n < x + ε$

Смысл точных граней для числовой последовательности схож со смыслом точных граней для множеств (см. [[1 семестр БИЛЕТЫ 1-28#Запись определений точных граней с помощью формул|выше]]).
### Определение бесконечно малой последовательности
[[1 семестр БИЛЕТЫ 1-28#Определение числовой последовательности|Числовая последовательность]] $\{a_n\}^{∞} _{n=1}$ называется бесконечно малой последовательностью (БМП), если

$$∀ ε > 0 : ∃ N = N(ε) : ∀n ≥ N: |a_n| < ε$$

то есть бесконечно много членов последовательности являются очень маленькими числами. Внутри $ε$-окрестности находится бесконечно много членов последовательности, а за её пределами — конечное число.

Запись $N = N(ε)$ означает, что $N$ меняется в зависимости от значения $ε$, т.е. $N$ зависит от $ε$.

Пример: $a_n = \frac{1}{n}, n∈\mathbb{N}$. Покажем, что эта последовательность является БМП.

Используем определение. $|\frac{1}{n}| < ε ⇒ \frac{1}{n} < ε$ 

Решаем это неравенство относительно n:

$n > \frac{1}{ε} ⇒ N(ε) = [\frac{1}{ε}] + 1$

Пусть $ε = 0.1 ⇒ N = N(ε) = 11; n = 12, 13, 14\dots$

$ε = 0.01 ⇒ N = N(ε) = 101$

$ε = 0.001 ⇒ N = N(ε) = 1001$ и т.д.

Пример: $a_n = \{1, \frac{1}{2}, 1, \frac{1}{3}, 1, \frac{1}{4}, 1, \frac{1}{5}, 1, \dots, \frac{1}{n}, 1, \dots\}$

Данная последовательность не является БМП, так как внутри ε-окрестности находится бесконечно много членов последовательности, за её пределами — тоже бесконечно много.
### Теорема об ограниченности БМП
**Любая БМП является [[1 семестр БИЛЕТЫ 1-28#Ограниченные последовательности|ограниченной]].**

Доказательство: пусть $\{a_n\}^{∞} _{n=1}$ — БМП. Покажем, что она является ограниченной. Проверим определение ограниченной последовательности:

$∃ m ∈ \mathbb{R} : ∀ n ∈ \mathbb{N} : a_n ≤ m$

$∀ ε > 0 : ∃ N = N(ε) : ∀ n ≥ N : |a_n| < ε$

Бесконечно много членов последовательности ограничено ε, а конечное число всегда ограничено $⇒ ∃ m ∈ \mathbb{R} : ∀ n ∈ \mathbb{N} : a_n ≤ m$, ч.т.д.
## Билет 4: Сумма бесконечно малых последовательностей, разность, алгебраическая сумма, произведение бесконечно малой последовательности на ограниченную последовательность
### Теорема о сумме двух БМП
**Сумма двух БМП есть БМП.**

Доказательство: пусть $\{a_n\}^{∞} _{n=1}, \{b_n\}^{∞} _{n=1}$ — БМП. Покажем, что $\{a_n+b_n\}^{∞} _{n=1}$ тоже БМП.

Используем определение БМП:

1. $∀ ε > 0 : ∃ N_1 = N_1(ε) : ∀ n ≥ N_1 : |a_n| < ε$
2. $∀ ε > 0 : ∃ N_2 = N_2(ε) : ∀ n ≥ N_2 : |b_n| < ε$

Складываем два неравенства и используем неравенство треугольника:

$$|a_n + b_n| ≤ |a_n| + |b_n| < \frac{ε}{2} + \frac{ε}{2} ⇒ |aₙ + bₙ| ≤ |aₙ| + |bₙ| < ε ⇒ |aₙ + bₙ| < ε$$

$∀ ε > 0 : ∀ n ≥ N(ε)$, где $N(ε)=max(N_1, N_2): |a_n + b_n| < ε ⇒ \{a_n+b_n\}^{∞} _{n=1}$ — БМП, ч.т.д.
### Теорема о разности двух БМП
**Разность двух БМП есть БМП.**

Доказательство: пусть $\{a_n\}^{∞} _{n=1}, \{b_n\}^{∞} _{n=1}$ — БМП. Покажем, что $\{a_n-b_n\}^{∞} _{n=1}$ тоже БМП.

$\{a_n-b_n\}^{∞} _{n=1}=\{a_n+(-b_n)\}^{∞} _{n=1}.$ Очевидно, что $\{-b_n\}^{∞} _{n=1}$ — БМП.

Так как сумма двух БМП есть БМП (см. [[1 семестр БИЛЕТЫ 1-28#Теорема о сумме двух БМП|выше]]), то $\{a_n-b_n\}^{∞} _{n=1}=\{a_n+(-b_n)\}^{∞} _{n=1}.$ — БМП, ч.т.д.

### Теорема о произведении БМП на ограниченную последовательность
**Произведение БМП на ограниченную последовательность есть БМП.**

Доказательство: пусть $\{a_n\}^{∞} _{n=1}$ — БМП, $\{b_n\}^{∞} _{n=1}$ — ограниченная последовательность. Покажем, что $\{a_n·b_n\}^{∞} _{n=1}$ — БМП.

Используем определение:
1. $∀ ε > 0 : ∃ N = N(ε) : ∀ n ≥ N : |a_n| < \frac{ε}{m}$
2. $∃ m ∈ \mathbb{R} : ∀ n ∈ \mathbb{N}: |b_n| ≤ m$

$|a_n·b_n| = |a_n|·|b_n| < \frac{ε}{m} · m = ε$

$|a_n·b_n|<ε, ∀ n ≥ N$ — это и есть определение БМП, ч.т.д.
### Теорема о произведении двух БМП
**Произведение двух БМП есть БМП.**

Доказательство: Любая БМП является ограниченной, поэтому данную теорему можно свести к [[1 семестр БИЛЕТЫ 1-28#Теорема о произведении БМП на ограниченную последовательность|теореме, описанной выше]].
## Билет 5: Связь бесконечно малой и бесконечно большой последовательности
### Определение бесконечно большой последовательности
[[1 семестр БИЛЕТЫ 1-28#Определение числовой последовательности|Числовая последовательность]] $\{b_n\}^{∞} _{n=1}$ называется бесконечно большой последовательностью (ББП), если

$$∀ M ∈ \mathbb{R} : ∃ N = N(M) : ∀n ≥ N(M) : |b_n| > M$$

, где M — по смыслу сколь угодно большое число. Таким образом, бесконечно много членов последовательности с достаточно большими номерами будут находиться вне интервала $(-M; M)$, а внутри интервала — лишь конечное число.

Пример: $b_n = n^2, n∈\mathbb{N}$. Покажем, что эта последовательность является ББП.

Используем определение. $|n^2| > M ⇒ n^2 > M$ 

Решаем это неравенство относительно n:

$n^2 > M ⇒ n > \sqrt{M} ⇒ N = N(M) = [\sqrt{M}] + 1$

$M = 100 ⇒ N = N(M) = 11$

$M = 10000 ⇒ N = N(M) = 101$ и т.д.
### Теорема о связи БМП и ББП
**Если $\{a_n\}^{∞} _{n=1}$ — БМП, то $\{\frac{1}{a_n}\}^{∞} _{n=1} (a ≠ 0)$ — ББП.**

Доказательство: Используем [[1 семестр БИЛЕТЫ 1-28#Определение бесконечно малой последовательности|определение БМП]]:

$$∀ ε > 0 : ∃ N = N(ε) : ∀n ≥ N : |a_n| < ε$$

$|\frac{1}{a_n}|=\frac{1}{|a_n|}>\frac{1}{ε}$. $M = \frac{1}{ε}$ ($M$ — сколь угодно большое число) $⇒ |\frac{1}{a_n}| > M.$

$N = N(ε), M = M(ε) ⇒ N = N(M)$

$∀n ≥ N(M) : |\frac{1}{a_n}| > M$, ч.т.д.
## Билет 6: Предел числовой последовательности. Необходимое и достаточное условие существования предела числовой последовательности. Теорема о единственности предела числовой последовательности
### Определение предела
[[1 семестр БИЛЕТЫ 1-28#Определение числовой последовательности|Числовая последовательность]] $\{a_n\}^{∞} _{n=1}$ сходится к числу $a$ ($\{a_n\} \xrightarrow{n→∞} a$) или имеет пределом число $a$ ($\lim _{n→∞} a _n = a$), если

$$
∀ ε > 0 : ∃ N = N(ε) : ∀n ≥ N(ε): |a_n - a| < ε
$$

т.е. члены последовательности будут близки к некоторому значению $a$, начиная с определенного номера $N$, зависящего от расстояния.

В $\varepsilon$-окрестности точки $a$ находится бесконечно много членов последовательности, а за её пределами — лишь конечное число ($\varepsilon$-окрестность точки $a$ — отрезок очень маленькой длины с центром в точке $a$).

Если $a = 0$, то получаем [[1 семестр БИЛЕТЫ 1-28#Определение бесконечно малой последовательности|определение БМП]].

$\{a_n\}^{∞} _{n=1}$ — БМП $⇒ \lim _{n→∞} a_n = 0$
#### Пример
$a_n = \frac{n}{n+1}, n ∈ \mathbb{N}.$ Доказать, что $\lim_{n→∞} a_n=1$.

Используем определение предела: $|\frac{n}{n+1}-1| < ε$

$$|\frac{n}{n+1}-1|=|\frac{n}{n+1}-\frac{n+1}{n+1}|=|\frac{n-n-1}{n+1}|=|\frac{-1}{n+1}|=|\frac{1}{n+1}|<ε$$

$$\frac{1}{n+1} < ε ⇒ n + 1 > \frac{1}{ε};\ N = N(ε) = [\frac{1}{ε}-1]+1$$

$ε = 0.1 ⇒ N = N(ε) = 10$

$ε = 0.01 ⇒ N = N(ε) = 100$
### Теорема о необходимом и достаточном условии существования предела
**Для существования предела $a$ числовой последовательности $\{a_n\}^{∞} _{n=1}$ необходимо и достаточно, чтобы она была представлена в виде (\*)  $a_n = a + \alpha_n$, где $\{\alpha_n\}$ — БМП.**

Доказательство необходимого условия: Пусть $\lim_{n→∞} a_n = a$. Покажем, что будет выполнять условие (\*).

Запишем определение предела для последовательности $\{a_n\}$:

$$
∀ ε > 0 : ∃ N = N(ε) : ∀n ≥ N(ε): |a_n - a| < ε
$$

Обозначим через $\alpha_n = a_n - a$, т.е. $a_n = a + \alpha_n$.

$|\alpha_n| < ε, ∀n ≥ N(ε) ⇒ \{\alpha_n\}$ — БМП. Условие (\*) выполняется, ч.т.д.

Доказательство достаточного условия: Пусть $a_n = a + \alpha_n$, где $\{\alpha_n\}$ — БМП. Покажем, что $\lim_{n→∞} a_n = a$.

Поскольку $\{\alpha_n\}$ — БМП, запишем [[1 семестр БИЛЕТЫ 1-28#Определение бесконечно малой последовательности|определение БМП]]:

$$
∀ ε > 0 : ∃ N = N(ε) : ∀n ≥ N(ε): |\alpha_n| < ε
$$

$\alpha_n = a_n - a ⇒ |\alpha_n - a| < ε, ∀n ≥ N(ε) ⇒ \lim_{n→∞} a_n = a$, ч.т.д.
### Теорема о единственности предела числовой последовательности
**Если любая числовая последовательность имеет предел, то только один.**

Доказательство:

Предположим противное: пусть числовая последовательность имеет два предела в точках $a$ и $b$.

Начертим $\varepsilon$-окрестности этих точек так, чтобы они не пересекались. По определению предела в $\varepsilon$-окрестности точки должно быть бесконечное количество членов последовательности, а за её пределами — конечное количество. Но если числовая последовательность имеет два предела, то в $\varepsilon$-окрестностях точек $a$ и $b$ будет бесконечное число членов последовательности.

Рассмотрим любую из этих точек. В пределах её $\varepsilon$-окрестности будет бесконечное количество членов последовательности, за её пределами тоже будет бесконечное количество членов последовательности, сконцентрированные в другой $\varepsilon$-окрестности. Это противоречит условию предела о том, что за пределами $\varepsilon$-окрестности бесконечного количества членов последовательности быть не может. Следовательно, числовая последовательность не может иметь больше одного предела, ч.т.д.
## Билет 7: Свойства пределов числовых последовательностей
### Теорема о сумме пределов
$$
∃ \lim_{n→∞} a_n = a, ∃ \lim_{n→∞} b_n = b ⇒ ∃ \lim_{n→∞} (a_n±b_n) = \lim_{n→∞} a_n ± \lim_{n→∞} b_n = a ± b
$$

Доказательство: Используем [[1 семестр БИЛЕТЫ 1-28#Теорема о необходимом и достаточном условии существования предела|необходимое и достаточное условие существования предела числовой последовательности]]:

$a_n = a + \alpha_n$, где $\{\alpha_n\}$ — БМП; $b_n = b + \beta_n$, где $\{\beta_n\}$ — БМП.

Сложим эти два равенства: $a_n + b_n = a + b + \alpha_n + \beta_n$.

$a_n + b_n$ — БМП по [[1 семестр БИЛЕТЫ 1-28#Теорема о сумме двух БМП|теореме о сумме двух БМП]].

Так как $\{a_n + b_n\} = a + b + БМП$, то $\lim_{n→∞} (a_n+b_n) = a + b = \lim_{n→∞} a_n + \lim_{n→∞} b_n$.

Доказательство для разности аналогично.
### Теорема о произведении пределов
$$
∃ \lim_{n→∞} a_n = a, ∃ \lim_{n→∞} b_n = b ⇒ ∃ \lim_{n→∞} (a_n · b_n) = \lim_{n→∞} a_n · \lim_{n→∞} b_n = a · b
$$

Доказательство: в силу необходимого и достаточного условия предела числовой последовательности $a_n = a + \alpha_n$, где $\{\alpha_n\}$ — БМП; $b_n = b + \beta_n$, где $\{\beta_n\}$ — БМП.

Находим произведение этих двух равенств: $a_n · b_n = (a + \alpha_n)(b + \beta_n) = ab + a\beta_n + \alpha_n b + \alpha_n\beta_n$

$a$ можно представить в виде ограниченной последовательности $⇒$ $a\beta_n$ — БМП [[1 семестр БИЛЕТЫ 1-28#Теорема о произведении БМП на ограниченную последовательность|как произведение БМП на ограниченную последовательность]].

Аналогично $\alpha_n b$ — БМП. $\alpha_n\beta_n$ — БМП как [[1 семестр БИЛЕТЫ 1-28#Теорема о произведении двух БМП|произведение двух БМП]].

[[1 семестр БИЛЕТЫ 1-28#Теорема об алгебраической сумме БМП|Сумма трёх БМП есть БМП]] $⇒$ $a_n · b_n = ab + БМП$ $⇒$ в силу необходимого и достаточного условия существования предела числовой последовательности $\lim_{n→∞} (a_n · b_n) = a · b = \lim_{n→∞} a_n · \lim_{n→∞} b_n$, ч.т.д.
### Теорема о частном пределов
$$
∃ \lim_{n→∞} a_n = a, ∃ \lim_{n→∞} b_n = b ⇒ ∃ \lim_{n→∞} \frac{a_n}{b_n} = \frac{\lim_{n→∞} a_n}{\lim_{n→∞} b_n} = \frac{a}{b} (b \ne 0; b_n \ne 0\ \forall n ∈ \mathbb{N})
$$

Доказательство: Покажем, что $\frac{a_n}{b_n}=\frac{a}{b} + \gamma_n$, где $\{\gamma_n\}$ — БМП.

$$
\gamma_n=\frac{a_n}{b_n}-\frac{a}{b}=\frac{a_n}{b_n}-\frac{a}{b}+\frac{a}{b_n}-\frac{a}{b_n}=\frac{a_n-a}{b_n}+\frac{ab-ab_n}{bb_n}=
$$

$$
=\frac{1}{b_n}(a_n-a)+\frac{a}{bb_n}(b-b_n)=\frac{1}{b_n}\alpha_n+\frac{a}{bb_n}(\beta_n)
$$

Докажем, что $\frac{1}{b_n}$ — ограниченная последовательность. Запишем определение предела для 

$$
∀ ε > 0 : ∃ N = N(ε) : ∀n ≥ N(ε): |b_n - b| < ε ⇒ b - ε < b_n < b + ε ⇒ $$

$$
⇒ \frac{1}{b - ε} > \frac{1}{b_n} > \frac{1}{b + ε} ⇒ \frac{1}{b + ε} < \frac{1}{b_n} < \frac{1}{b - ε} ⇒ |\frac{1}{b_n}| < \frac{1}{b - ε} \forall n ∈ \mathbb{N}(ε)
$$

Бесконечное количество членов искомой последовательности ограничено, а оставшиеся члены в любом случае будут ограничены, следовательно, $\frac{1}{b _n}$ — ограниченная последовательность.

Следовательно, слагаемые $\frac{1}{b_n}\alpha_n$ и $\frac{a}{bb_n}(\beta_n)$ являются БМП как произведения БМП на ограниченную последовательность.

$\frac{1}{b}\alpha_n+\frac{a}{bb_n}(\beta_n)$ — БМП, то есть $\{\gamma_n\}$ — БМП $⇒$ $\frac{a_n}{b_n}=\frac{a}{b} + \gamma_n$, где $\{\gamma_n\}$ — БМП.

Следовательно, в силу необходимого и достаточного условия существования предела числовой последовательности $\lim_{n→∞} \frac{a_n}{b_n} = \frac{a}{b} = \frac{\lim_{n→∞} a_n}{\lim_{n→∞} b_n}$. ч.т.д.
## Билет 8: Предельный переход в неравенствах числовых последовательностей
### Теоремы о предельном переходе в неравенствах
$$
∃ \lim_{n→∞} a_n; \forall n ∈ \mathbb{N}: a_n \le b ⇒ \lim_{n→∞} a_n \le b
$$

Доказательство: Обозначим через $a$ $\lim_{n→∞} a_n$. Предположим противное: $a > b$.

Начертим на координатной прямой точки $a$ и $b$ с $\varepsilon$-окрестностью около точки $a$ так, что $b < a - ε$. В $\varepsilon$-окрестности точки $a$ находится бесконечно много членов исходной последовательности с достаточно большими номерами, а за её пределами — лишь конечное число. Очевидно, что не все члены последовательности меньше $b$. Получаем противоречие.

Следовательно, $a \le b$, ч.т.д.
### Теорема
**Пусть есть сходящаяся последовательность $\{a_n\}^{∞} _{n=1},\forall n ∈ \mathbb{N} : a_n \ge c$. Тогда $\lim _{n→∞} a_n \ge c$.**

Доказательство аналогично предыдущей теореме.
### Теорема 
**Пусть есть две сходящиеся последовательности $\{a_n\}^{∞} _{n=1}$ и $\{b_n\}^{∞} _{n=1}$ такие, что $a_n \le b_n, \forall n ∈ \mathbb{N}$. Тогда $\lim _{n→∞} a_n \le \lim_{n→∞} b_n$.**

Доказательство: Пусть $\{c_n\}^{∞} _{n=1}$ — последовательность, задающаяся как $\{a_n - b_n\}$. Тогда

$$
a_n \le b_n, \forall n ∈ \mathbb{N} ⇒ a_n - b_n \le 0 ⇒ c_n \le 0
$$

В силу теоремы о [[1 семестр БИЛЕТЫ 1-28#Теоремы о предельном переходе в неравенствах|предельном переходе в неравенствах]] $\lim_{n→∞} c_n \le 0$. Следовательно, имеет место следующее:

$$
\lim_{n→∞} c_n \le 0 ⇒ \lim_{n→∞}(a_n - b_n) \le 0 ⇒ \lim_{n→∞} a_n - \lim_{n→∞} b_n \le 0 ⇒ \lim_{n→∞} a_n \le \lim_{n→∞} b_n
$$

, ч.т.д.
### Лемма о двух милиционерах
**Пусть есть три последовательности $\{a_n\}^{∞} _{n=1}$, $\{b_n\}^{∞} _{n=1}$ и $\{c_n\}^{∞} _{n=1}$ такие, что $a_n \le b_n \le c_n, \forall n ∈ \mathbb{N}$ и $\lim _{n→∞} a_n = \lim _{n→∞} c_n = c$. Тогда $\lim _{n→∞} b _n = c$.**

Доказательство: Используем [[1 семестр БИЛЕТЫ 1-28#Определение предела|определение предела]] для последовательностей $\{a_n\}$ и $\{c_n\}$:

$$
∀ ε > 0 : ∃ N_a = N_a(ε) : ∀n ≥ N_a(ε): |a_n - c| < ε
$$

$$
∀ ε > 0 : ∃ N_c = N_c(ε) : ∀n ≥ N_c(ε): |c_n - c| < ε
$$

Бесконечно много членов последовательности $\{b_n\}$ оказываются в $\varepsilon$-окрестности точки $c$, а за её пределами — лишь конечное число. Следовательно, для последовательности $\{b_n\}$ выполнено определение предела $∀ ε > 0 : ∃ N_b = N_b(ε) : ∀n ≥ N_b(ε): |b_n - c| < ε$, где $N_b(ε) = max(N_a, N_c)$, ч.т.д.
## Билет 9: Теорема Вейерштрасса (для монотонной последовательности)
### Определение монотонной последовательности
[[1 семестр БИЛЕТЫ 1-28#Определение числовой последовательности|Числовая последовательность]] $\{a_n\}^{∞} _{n=1}$ называется монотонно возрастающей (или убывающей), если $a _{n+1} \ge (\le)\ a _n, \forall n ∈ \mathbb{N}$ (каждый следующий член больше или меньше предыдущего).

Если $a_{n+1} > (<)\ a_n, \forall n ∈ \mathbb{N}$, то последовательность называют строго монотонно возрастающей (или убывающей).

Если числовая последовательность монотонно возрастает или убывает, начиная с некоторого номера N, то такую последовательность называют монотонно возрастающей или убывающей, начиная с номера N.
#### Примеры
$$
x_n = \frac{1}{n}, n ∈ \mathbb{N};\ x_{n+1} = \frac{1}{n+1}
$$

$$
\frac{x_{n+1}}{x_n} = \frac{n}{n+1} < 1, \forall n ∈ \mathbb{N} ⇒ x_{n+1} < x_n, \forall n ∈ \mathbb{N}
$$

$x_n$ — монотонно убывающая последовательность.

$$
x_n = \frac{n}{n + 1}, n ∈ \mathbb{N};\ x_{n+1}=\frac{n+1}{n+2}
$$

$$
\frac{x_{n+1}}{x_n} = \frac{n+1}{n+2} · \frac{n+1}{n} = \frac{n^2 + 2n + 1}{n^2 + 2n} > 1, \forall n ∈ \mathbb{N}
$$

$x_n$ — монотонно возрастающая последовательность.

$$
x_n = n^2, n ∈ \mathbb{N};\ x_{n+1} = (n+1)^2 = n^2 + 2n + 1
$$

$$
x_{n+1}-x_n = n^2 + 2n + 1 - n^2 = 2n + 1 > 0, \forall n ∈ \mathbb{N}
$$

$x_n$ — монотонно возрастающая последовательность.
### Теорема Вейерштрасса для монотонной последовательности
**Любая монотонно возрастающая (убывающая) [[1 семестр БИЛЕТЫ 1-28#Ограниченные последовательности|ограниченная]] последовательность имеет предел (причём этот предел равен супремуму (инфимуму) последовательности).**

Доказательство: Докажем теорему для монотонно возрастающей ограниченной последовательности. Для монотонно убывающей ограниченной последовательности доказательство аналогично.

Возьмём последовательность $\{a_n\}$, которая монотонно возрастает до определённого числа $a$. Последовательность является ограниченной, следовательно, у неё есть точная верхняя грань a = sup $\{a_n\}$.

Запишем определение [[1 семестр БИЛЕТЫ 1-28#Ограниченные последовательности|супремума]]:
1. $∀ n ∈ \mathbb{N}: a_n ≤ a$
2. $∀ ε > 0 : ∃ n ∈ \mathbb{N} : a - ε < a_n < a$

Используем тот факт, что $a_n$ — монотонно возрастающая последовательность. Следовательно,

$$
a_{N} < a_{N+1} < a_{N+2} < ... < a, \forall n > N ⇒ |a_n - a| < ε
$$

Выполнено определение предела $\lim_{n→∞} a_n = a$, ч.т.д.
## Билет 10: Теорема о вложенных отрезках
### Теорема о вложенных промежутках (отрезках)
**Пусть есть бесконечная система отрезков $\{[a_n, b_n]\}^{∞} _{n=1}$, вложенных друг в друга ($[a _{n+1}, b _{n+1}] < [a _n, b _n], \forall n ∈ \mathbb{N}$), причём длины этих отрезков стремятся к нулю ($\lim _{n→∞} (b _n - a _n) = 0$). Тогда будет существовать единственная точка $c$, которая принадлежит сразу всем отрезкам.**

Доказательство: Рассмотрим последовательности $\{a_n\}$ и $\{b_n\}$. $\{a_n\}$ монотонно возрастает, $\{b_n\}$ монотонно убывает в силу условия теоремы.

$a_1 \le a_n \le b_n, a_n \le b_n \le b_1, \forall n ∈ \mathbb{N} ⇒$ обе последовательности ограничены.

В силу теоремы Вейерштрасса существует предел $\{a_n\}$ и $\{b_n\}$.

$$
\lim_{n→∞} (b_n - a_n) = 0 ⇒ \lim_{n→∞} b_n - \lim_{n→∞} a_n = 0 ⇒ \lim_{n→∞} b_n = \lim_{n→∞} a_n = c
$$

в силу условия сходимости длин и [[1 семестр БИЛЕТЫ 1-28#Теорема о сумме пределов|свойства пределов]].

В силу теоремы Вейерштрасса $c$ = sup $a_n$ и $c$ = inf $b_n$. Это означает, что $c$ принадлежит сразу всем отрезкам.

Покажем теперь, что эта точка является единственной. Предположим противное: пусть существует точка $a \ne c$, которая тоже принадлежит сразу всем отрезкам.

Выберем $\varepsilon$-окрестность точки $a$ так, чтобы $c$ не входила в неё. В $\varepsilon$-окрестности точки $a$ лежит бесконечно много членов последовательности $a_n$ и $b_n$, это противоречит тому, что $c$ — предел последовательности $\{a_n\}$ и $\{b_n\}$, ч.т.д.
## Билет 11: Число e
### Теорема
**Предел последовательности $x_n = (1+\frac{1}{n})^n, n ∈ \mathbb{N}$ существует и равен числу $e = 2.71828...$**

Доказательство: Докажем, что последовательность $x_n$ является [[1 семестр БИЛЕТЫ 1-28#Определение монотонной последовательности|монотонно возрастающей]] и [[1 семестр БИЛЕТЫ 1-28#Ограниченные последовательности|ограниченной]].

$$
(a+b)^n = a^n + \frac{n}{1!}a^{n-1}b + \frac{n(n-1)}{2!}a^{n-2}b^2 + \frac{n(n-1)(n-2)}{3!}a^{n-3}b^3 + \dots + b^n
$$

$$
x_n = (1+\frac{1}{n})^n = 1 + \frac{n}{1!}\frac{1}{n} + \frac{n(n-1)}{2!}\frac{1}{n^2} + \frac{n(n-1)(n-2))}{3!}\frac{1}{n^3} + \dots + \frac{1}{n^n}
$$

Выразим некоторые слагаемые:

$$
\frac{n(n-1)}{2!}\frac{1}{n^2}=\frac{n^2-n}{2!n^2}=\frac{n-1}{2!n}=\frac{1}{2!}(1 - \frac{1}{n})
$$

$$
\frac{n(n-1)(n-2)}{3!}\frac{1}{n^3} = \frac{1}{3!}\frac{n(n-1)(n-2)}{n·n·n}=\frac{1}{3!}(1-\frac{1}{n})(1-\frac{2}{n})
$$

Перепишем формулу $x_n$:

$$
x_n = 2 + \frac{1}{2!}(1-\frac{1}{n}) + \frac{1}{3!}(1-\frac{1}{n})(1 - \frac{2}{n}) + \dots + \frac{1}{n^n}
$$

Запишем эту же формулу для $x_{n+1}$:

$$
x_{n+1} = 2 + \frac{1}{2!}(1-\frac{1}{n+1}) + \frac{1}{3!}(1-\frac{1}{n+1})(1 - \frac{2}{n+1}) + \dots + \frac{1}{(n+1)^{n+1}}
$$

Можно увидеть, что второе слагаемое члена $x_{n+1}$ больше, чем у члена $x_n$. То же самое можно сказать про все остальные слагаемые, кроме последнего. Последнее слагаемое уменьшается при увеличении $n$, но его увеличением можно пренебречь при достаточно больших $n$. Следовательно, $x_{n+1} > x_n, \forall n ∈ \mathbb{N} ⇒ {x_n}$ — монотонно возрастающая последовательность.

Покажем, что $\{x_n\}$ — ограничена. Запишем оценку сверху для $x_n$:

$$
x_n < 2 + \frac{1}{2!} + \frac{1}{3!} + \dots + \frac{1}{n!}
$$

$$
\frac{1}{3!} = \frac{1}{1·2·3} < \frac{1}{2^2};\ \frac{1}{4!} = \frac{1}{1·2·3·4} < \frac{1}{2^3} ⇒ x_n < 2 + \frac{1}{2} + \frac{1}{2^2} + \frac{1}{2^3} + \dots + \frac{1}{2^n}
$$

Вычислим сумму бесконечной геометрической прогрессии, вспомнив формулу суммы для первых $n$ её членов:

$$
S_n = \frac{b_1(q^n-1)}{q-1};\ b_1 = \frac{1}{2};\ q = \frac{1}{2};\ \lim_{n→∞} \frac{\frac{1}{2}((\frac{1}{2})^n-1)}{\frac{1}{2}-1} = \lim_{n→∞} \frac{\frac{1}{2}(\frac{1}{2^n}-1)}{-\frac{1}{2}} = \frac{\frac{1}{2}·(-1)}{-\frac{1}{2}} = 1
$$

Или можно использовать формулу для вычисления бесконечно убывающей геометрической прогрессии:

$$
S = \frac{b_1}{1 - q};\ S = \frac{\frac{1}{2}}{1-\frac{1}{2}} = 1
$$

С помощью [[1 семестр БИЛЕТЫ 1-28#Теоремы о предельном переходе в неравенствах|предельного перехода в неравенствах]] получим:

$$
x_n < 3 ⇒ \lim_{n→∞} x_n < 3
$$
## Билет 12: Теорема о существовании предельной точки ограниченной последовательности
### Определение предельных точек
Точка $a$ называется предельной точкой исходной последовательности, если в любой сколь угодно малой её $\varepsilon$-окрестности находится бесконечно много членов исходной последовательности.

Принципиальное отличие предельной точки от [[1 семестр БИЛЕТЫ 1-28#Определение предела|предела]] заключается в том, что за пределами $\varepsilon$-окрестности предельной точки может быть также бесконечно много членов исходной последовательности.

Пример: $x_n = (-1)^n, \forall n ∈ \mathbb{N}$. Выделим две подпоследовательности:

$$
\{x_{2n}\} = 1, \forall n ∈ \mathbb{N}; \{x_{2n-1}\} = -1, \forall n ∈ \mathbb{N} ⇒ x_{2n-1} \xrightarrow{n→∞}-1; x_{2n} \xrightarrow{n→∞}1;
$$

Точки -1 и 1 являются предельными для исходной последовательности, хотя сама последовательность является расходящейся.
### Теорема
**Любая [[1 семестр БИЛЕТЫ 1-28#Ограниченные последовательности|ограниченная последовательность]] имеет хотя бы одну предельную точку.**

Доказательство: Пусть $\{x_n\}$ — ограниченная. Запишем определение ограниченной последовательности:

$$∃ m ∈ \mathbb{R} : ∀ n ∈ \mathbb{N}: |a_n| ≤ m$$

Делим отрезок $[-M; M]$ пополам, выбирая ту половину, которая содержит бесконечно много членов исходной последовательности. Снова делим её пополам и снова выбираем ту половину, где содержится бесконечно много членов исходной последовательности, и т.д. Получим бесконечную систему вложенных отрезков. Длина $k$-го отрезка = $\frac{2M}{2^k} \xrightarrow{k→∞} 0$, т.е. длины отрезков стремятся к нулю. Тогда [[1 семестр БИЛЕТЫ 1-28#Теорема о вложенных промежутках (отрезках)|по теореме о вложенных отрезках]] существует единственная точка, которая принадлежит сразу всем отрезкам. В силу нашего построения каждый отрезок содержит бесконечно много точек исходной последовательности, следовательно, найденная точка является предельной, ч.т.д.
## Билет 13: Теорема Больцано-Вейерштрасса. Лемма Бореля
### Определение подпоследовательности
Пусть есть [[1 семестр БИЛЕТЫ 1-28#Определение числовой последовательности|числовая последовательность]] $\{x_n\}^{∞} _{n=1}$ с пронумерованными элементами $x_1,\ x_2,\ x_3,\ \dots,\ x_n,\ \dots$

Выберем из неё произвольным образом любое количество элементов, которые обозначим следующим образом $x_{n_1},\ x_{n_2},\ x_{n_3},\ \dots, x_{n_k},\ \dots$, где $n_1,\ n_2,\ n_3,\ \dots,\ n_k,\ \dots$ — порядковые номера в новой последовательности.

Новая последовательность называется подпоследовательностью исходной последовательности и обозначается $\{x_{n_k}\}^{∞} _{n=1}$.
### Теорема
**Если последовательность $\{x_n\}^{∞} _{n=1}$ имеет предел $a$, то она имеет ровно одну предельную точку.**

Доказательство: Предположим противное. Пусть последовательность $\{x_n\}^{∞} _{n=1}$ имеет ещё одну предельную точку $b \ne a$. Выберем $\varepsilon$-окрестности точек $a$ и $b$ так, чтобы они не пересекались.

В $\varepsilon$-окрестности точки $a$ находится бесконечно много членов исходной последовательности, за её пределами — конечное число, что противоречит тому, что точка $b$ является предельной.

**ПРИМЕЧАНИЕ АВТОРА**: смысл доказательства в том, чтобы доказать, что у сходящейся последовательности не может быть больше одной предельной точки. Разница между предельной точкой и пределом в том, что у предела за пределами $\varepsilon$-окрестности необходимой точки не может быть бесконечного количества членов, а у предельной точки может (см. пример выше).
### Теорема
**Если последовательность имеет предельную точку, то из неё можно выделить подпоследовательность, сходящуюся к этой точке.**

Доказательство: Пусть точка $a$ является предельной точкой последовательности $\{x_n\}^{∞} _{n=1}$. Рассмотрим БМП $\{\varepsilon_n\}^{∞} _{n=1}$, которая сформирована из $\varepsilon$-окрестности точки $a$. (например, $\varepsilon = \frac{1}{n}, n ∈ \mathbb{N}$).

Выберем элемент $x_{n_1} ∈ (a - \varepsilon_1; a + \varepsilon_1); x_{n_2} ∈ (a - \varepsilon_2; a + \varepsilon_2), n_2 > n_1$ и т.д. до $x_{n_k} ∈ (a - \varepsilon_k; a + \varepsilon_k), n_k > n_{k-1}$.

$$
|x_{n_k}-a|<\varepsilon_k, \forall k\ \forall \varepsilon_k > 0\ \forall n \ge n_k ⇒ x_{n_k} \xrightarrow{n→∞} a
$$

, ч.т.д.

См. две предыдущие теоремы для доказательства.
### Лемма Бореля
**Пусть есть отрезок $[a; b]$, каждая точка которого принадлежит одному из бесконечного количества интервалов вида $I_\alpha = (c_\alpha, d_\alpha)$. Тогда можно выделить конечное число интервалов, покрывающих данный отрезок ($[a, b] ⊂ I_{\alpha_1} ⋃ I_{\alpha_2} ⋃ \dots ⋃ I_{\alpha_n}$).**

Доказательство: Предположим противное: нельзя выделить конечное число интервалов, покрывающих отрезок $[a; b]$.

Разделим $[a; b]$ пополам и выберем ту половину, которую нельзя покрыть конечным числом интервалов. Снова разделим её пополам и снова выберем ту половину, которую нельзя покрыть конечным числом интервалов. Получим бесконечную систему вложенных отрезков, длины которых стремятся к нулю. ## Билет 15: Условие Коши сходимости числовой последовательности
### Условие Коши сходимости числовой последовательности
Замечание: при исследовании сходимости числовой последовательности мы использовали определение предела, при этом необходимо было знать само значение предельной точки. Но это не всегда возможно, поэтому существенно усложняется анализ сходимости. Условие Коши позволяет провести анализ сходимости, не зная самого значения предела.

**Числовая последовательность $\{x_n\}^{∞} _{n=1}$ называется фундаментальной, или удовлетворяющей условию Коши, если выполняется условие**

$$
\forall ε>0 : \exists N = N(ε) : \forall n \ge N(ε), p ∈ \mathbb{N} : |x_{n+p} - x_n| < ε
$$

Таким образом, для фундаментальной последовательности, начиная с некоторого, достаточно большого номера $N$, бесконечно много членов последовательности находятся на бесконечно малом расстоянии друг от друга.
### Теорема Коши о сходимости числовой последовательности
**Для того чтобы числовая последовательность являлась сходящейся, необходимо и достаточно, чтобы она удовлетворяла условию Коши.**

Доказательство необходимого условия Пусть числовая последовательность является сходящейся. Покажем, что она удовлетворяет условию Коши.

Запишем [[1 семестр БИЛЕТЫ 1-28#Определение предела|определение предела]]:

$$
∀ ε > 0 : ∃ N = N(ε) : ∀n ≥ N(ε): |x_n - a| < ε
$$

Рассмотрим произвольный элемент последовательности $x_m$, где $m=n+p, p ∈ \mathbb{N}$. $m=n+p > n > N(ε)$. Следовательно, $x_m$ принадлежит $\varepsilon$-окрестности точки $a$; $|x_m - x_n| < 2ε$; условие Коши выполняется для всех $m > N(ε)$, ч.т.д.

Доказательство достаточного условия: пусть выполняется условие Коши. Покажем, что последовательность является сходящейся.

(Я доказывал это самостоятельно. Если есть какие-то неточности, свяжитесь со мной, я постараюсь их исправить)

Возьмём $\varepsilon$-окрестность точки $x_n$. Тогда при любом $p$ $|x_{n+p} - x_n|<ε$, то есть все члены $x_{n+p}, \forall p ∈ \mathbb{N}$ входят в $\varepsilon$-окрестность точки $x_n$. Следовательно, начиная с номера $N(ε)$, бесконечно много членов последовательности находятся в $\varepsilon$-окрестности точки $x_n$, эта точка будет являться пределом, ч.т.д.
## Билет 16: Способы задания функций. Предел функции
### Определение функции
Функция — правило, закон, отображение, по которому каждому значению $x$ из $D$, где $D$ — область существования функций, ставят в соответствие значение $y$ из $E$, где $E$ — область значения функции.

Способы задания функций:

1. аналитический
2. графический
3. табличный (функция имеет дискретный вид)
4. с помощью логических выражений

Примеры функций, заданных логическими выражениями

$$

2. функция Дирихле:

$$
D(x) = 
\begin{cases}
1, x ∈ \mathbb{Q}\\
0, x ∈ \mathbb{I}
\end{cases}
$$

где $\mathbb{Q}$ — множество рациональных чисел, $\mathbb{I}$ — множество иррациональных чисел.

R(x) =

### Определение точки сгущения
Точка $a$ [[1 семестр БИЛЕТЫ 1-28#Операции принадлежности|∈]] $D$ называется точкой сгущения, если в её любой сколько угодно малой $\varepsilon$-окрестности находится бесконечно много точек этого множества.
### Определение предела функции по Гейне (на языке последовательности)
Пусть $a$ — [[1 семестр БИЛЕТЫ 1-28#Определение точки сгущения|точка сгущения]]. Говорят, что $\lim_{x→a} f(x) = b$, если

$$
\forall \{x _n\} \xrightarrow{n→∞}a ⇒ \{f(x _n)\} \xrightarrow{n→∞} b
$$

Данное определение описывает предел функции с помощью определения [[1 семестр БИЛЕТЫ 1-28#Определение предела|предела для числовой последовательности]].
### Определение предела функции по Коши (на языке ε — δ)
Пусть $a$ — точка сгущения. Говорят, что $\lim_{x→a} f(x) = b$, если

$$
\forall ε > 0 : \exists \delta = \delta(\varepsilon) : |x-a| < \delta, |f(x)-b| < \varepsilon
$$

При сужении $\delta$-окрестности сузится и $\varepsilon$-окрестность; значение $f(x)$ будет всё ближе к $b$.

### Теорема об эквивалентности определений предела функции
**Определения по Коши и по Гейне идентичны. Если функция имеет предел в смысле одного определения, то она имеет предел в смысле другого.**

Доказательство:

Пусть верно определение по Коши. Докажем, что определение по Гейне тоже верно.

$$

$$

Рассмотрим последовательность $\forall \{x _n\} → a$. Подставим $x _n$ вместо $x$:

$$
|x_n - a| < \delta, |f(x_n) - b| < \varepsilon ⇒ \lim_{n→∞} f(x_n) = b,
$$

выполняется определение предела по Гейне.

Пусть верно определение по Гейне. Докажем, что определение по Коши тоже верно.

Предположим противное:

$$

$$
|x_n - a| < \delta ⇒ |f(x_n) - b| > \varepsilon_0
$$

Получили противоречие с определением предела по Гейне:

$$

## Билет 19: Теорема Коши о промежуточном значении непрерывной функции. Следствие
### Определение непрерывной функции
[[1 семестр БИЛЕТЫ 1-28#Определение функции|Функция]] $y = f(x)$ называется непрерывной в точке $a$, если $\lim _{x→a} f(x) = f(a)$, т.е. [[1 семестр БИЛЕТЫ 1-28#Определение предела функции по Коши (на языке ε — δ)|предел функции]] в [[1 семестр БИЛЕТЫ 1-28#Определение точки сгущения|точке сгущения]] равен значению в этой точке.

На языке $\varepsilon-\delta$:

$$
\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : |x-a|<\delta, |f(x) - f(a)| < \varepsilon
$$

Функция $y = f(x)$ непрерывна на [[1 семестр БИЛЕТЫ 1-28#Определение множества|множестве]] $D$, если она непрерывна в каждой точке этого множества.
### Теорема Коши о промежуточном значении непрерывной функции
**Пусть функция $y = f(x) ∈ C [a; b]$ (функция $f(x)$ непрерывна на отрезке $[a; b]$); на концах отрезков функция принимает противоположные по знаку значения ($f(a)f(b) < 0$). Тогда существует единственная точка $c$ такая, что $f(c) = 0$.**

Доказательство: Разделим отрезок $[a; b]$ пополам и выберем ту половину, в которой функция на концах отрезка имеет противоположные по знаку значения. Снова делим эту половину пополам и снова выбираем ту часть, где функция на концах отрезка принимает противоположные значения, и т.д. пока не достигнем точности $\varepsilon$. Построена система вложенных отрезков, длины которых стремятся к нулю. Обозначим эти отрезки как $\{[\alpha _n, \beta _n]\}^{∞} _{n=1}$. Тогда в силу [[1 семестр БИЛЕТЫ 1-28#Теорема о вложенных промежутках (отрезках)|теоремы о вложенных отрезках]] существует единственная точка $c : f(c) = 0$.

Обозначим через $\alpha _n$ тот конец отрезка, где $f(x) \ge 0$, а через $\beta _n$ — тот конец отрезка, где $f(x) \le 0$. Очевидно, что $\alpha _n \xrightarrow{n→∞} c$, $\beta _n \xrightarrow{n→∞} c$. В силу непрерывности функции $f(x)$ и [[1 семестр БИЛЕТЫ 1-28#Теоремы о предельном переходе в неравенствах|теоремы о предельном переходе в неравенствах]]:

$$
\alpha _n \xrightarrow{n→∞} c; f(\alpha _n) \ge 0 ⇒ \lim _{n→∞} f(\alpha _n) = f(c) \ge 0
$$

$$
\beta _n \xrightarrow{n→∞} c; f(\beta _n) \le 0 ⇒ \lim _{n→∞} f(\beta _n) = f(c) \le 0
$$

$$
f(c) \ge 0; f(c) \le 0 ⇒ f(c) = 0, \text{ч.т.д.}
$$

### Следствие из теоремы Коши о промежуточном значении непрерывной функции
**Пусть $f(a) = d, f(b) = e$, а $f(x)$ непрерывна на отрезке $[a; b]$. Тогда значения функции непрерывно заполняют отрезок $[d; e]$.**

Доказательство: Рассмотрим функцию $y = f(x) - g$, где $g ∈ [d; e]$ (т.е. $d < g < e$). Надо доказать, что существует точка $c ∈ [a; b]$ такая, что $f(c) = g$.

Найдем значения функции $y(x)$ на концах отрезка:

$$
y(a) = f(a) - g = d - g; d < g < e ⇒ d - g < 0
$$

$$
y(b) = f(b) - g = e - g; d < g < e ⇒ e - g > 0
$$

Функция $y(x)$ является непрерывной как сумма двух непрерывных функций $f(x)$ и $-g$. В силу теоремы Коши существует точка $c ∈ [a; b]$ такая, что $y(c) = 0$.

$$
y(c) = f(c) - g = 0 ⇒ f(c) = g
$$

В силу произвольности выбора точек $c$ и $g$ это утверждение справедливо для всего промежутка $[a; b]$ и $[d; e]$, ч.т.д.
## Билет 20: Сложная функция. Теорема о непрерывности сложной функции
### Определение  сложной функции
Пусть [[1 семестр БИЛЕТЫ 1-28#Определение функции|функция]] $y = f(x)$ определена на [[1 семестр БИЛЕТЫ 1-28#Определение множества|множестве]] $D$. Пусть $R$ — область значений этой функции, причём $R _f ⊂ D$. Пусть функция $z = g(y)$ определена на множестве $D$. Тогда можно задать сложную функцию или суперпозицию двух функций $z = g(f(x))$ или $z = g∙f$, где $f(x)$ — внутренняя функция, а $g(x)$ — внешняя функция.
#### Примеры
1. $y = \sin(\cos x)$
2. $y = e^{ln(x)} = \exp(\ln(x))$
3. $y = x^{x^{x}}$

Колмогоров доказал, что любую функцию можно представить как суперпозицию многих элементарных функций.
### Теорема о непрерывности сложной функции
**Пусть фукнция $y = f(x)$ непрерывна в точке $x _0$, $z = g(y)$ непрерывна в точке $y _0$. Тогда $z = g(f(x))$ будет непрерывна в точке $x _0$.**

Доказательство: Перейдём на язык последовательностей и используем [[1 семестр БИЛЕТЫ 1-28#Определение предела функциипо Гейне (на языке последовательности)|определение предела по Гейне]]. Нам нужно доказать, что $\forall\{x _n\} \xrightarrow{n→∞} x _0 ⇒ \{g(f(x _n))\} \xrightarrow{n→∞} g(f(x _0))$.

В силу непрерывности внутренней функции: $\forall\{x _n\} \xrightarrow{n→∞} x _0 ⇒ \{f(x _n)\} \xrightarrow{n→∞} f(x _0)$

Обозначим через $y _n = f(x _n)$. В силу непрерывности внешней функции $\forall\{y _n\} \xrightarrow{n→∞} y _0  ⇒ \{g(y _n)\} \xrightarrow{n→∞} g(y _0) ⇒ \{g(f(x _n))\} \xrightarrow{n→∞} g(f(x _0))$, ч.т.д.
## Билет 21: Понятие монотонной функции. Критерий непрерывности для монотонной функции
### Определение монотонной функции
Пусть [[1 семестр БИЛЕТЫ 1-28#Определение функции|функция]] $y = f(x)$ определена на [[1 семестр БИЛЕТЫ 1-28#Определение множества|множестве]] $D$. Функция называется монотонно возрастающей на множестве $D$, если $\forall x _1, x _2 ∈ D : x _1 < x _2 ⇒ f(x _1) \le f(x _2)$, Если знак строгий ($f(x _1) < f(x _2)$), тогда функция строго монотонно возрастает.

Функция называется монотонно убывающей на множестве $D$, если $\forall x _1, x _2 ∈ D : x _1 < x _2 ⇒ f(x _1) \ge f(x _2)$, Если знак строгий ($f(x _1) > f(x _2)$), тогда функция строго монотонно убывает.

Примеры:
1. $y = x^2$. монотонно возрастает на $D ∈ [0; +∞)$; монотонно убывает на $D ∈ (-∞; 0]$.
2. $y = c = const$. монотонно возрастает и убывает на $D ∈ (-∞; +∞)$.
3. $y = \sin x$. монотонно возрастает на $D ∈ [-\frac{\pi}{2} + 2\pi n; \frac{\pi}{2} + 2\pi n], n ∈ \mathbb{Z}$; монотонно убывает на $D ∈ [\frac{\pi}{2} + 2\pi n; \frac{3\pi}{2} + 2\pi n], n ∈ \mathbb{Z}$.
### Критерий непрерывности для монотонной функции
**Пусть функция $y = f(x)$ монотонно возрастает или монотонно убывает на промежутке $[a; b]$. На концах отрезка функция принимает значения $f(a) = \alpha$, $f(b) = \beta$. Для того чтобы эта функция являлась непрерывной, необходимо и достаточно, чтобы её значения полностью заполняли промежуток $[\alpha; \beta]$.**

Для определённости будем полагать, что функция $y = f(x)$ монотонно возрастает.

Доказательство необходимого условия: Пусть функция непрерывна. Тогда в силу [[1 семестр БИЛЕТЫ 1-28#Следствие из теоремы Коши о промежуточном значении непрерывной функции|следствия из теоремы Коши]] её значения полностью заполняют промежуток $[\alpha; \beta]$.

Доказательство достаточного условия: Возьмём произвольное достаточно малое значение $\varepsilon$. В силу того, что значения функции полностью заполняют промежуток $[\alpha; \beta]$:

$$
\exists x _1 ∈ [\alpha; \beta] : f(x _1) = f(x _0) - \varepsilon; \exists x _2 ∈ [\alpha; \beta] : f(x _2) = f(x _0) + \varepsilon
$$

Возьмём $x$ такой, что $x _1 < x < x _2$. В силу монотонности функции $f(x _1) < f(x) < f(x _2) ⇒ f(x _0) - \varepsilon < f(x) < f(x _0) + \varepsilon ⇒$ $|f(x) - f(x _0)| < \varepsilon ⇒ |x - x _0| < \delta, \delta = \delta(\varepsilon)$.

Получили [[1 семестр БИЛЕТЫ 1-28#Определение предела функции по Коши (на языке ε — δ)|определение предела функции]] на языке $\varepsilon-\delta$:

$$
\forall ε > 0 : \exists \delta = \delta(\varepsilon) : |x-x _0| < \delta, |f(x)-f(x _0)| < \varepsilon
$$

Выполняется [[1 семестр БИЛЕТЫ 1-28#Определение непрерывной функции|определение непрерывности функции в точке]]: $\lim _{x→x _0} f(x) = f(x _0)$. Поскольку точки $x$ и $x _0$ была выбрана нами произвольно, это будет справедливо для любой точки отрезка $[a; b]$.
## Билет 22: Обратная функция. Теорема об обратной функции
### Определение
Пусть функция $y = f(x)$ определена на множестве $D$  и пусть выполняется условие

$$
\forall x ∈ D\ \exists! y : f(x) = y 
$$

т.е. каждому значению $x$ из области определения $D$ соответствует единственное значение $y$ (условие однозначности соответствия). Тогда для этой функции можно определить или задать обратную функцию $x = f^{-1}(y)$.

Пример: для функции $y = x^2, x ∈ (-∞; +∞)$. Для этой функции обратной будет являться функция $x = -\sqrt{y}$ при $x ∈ (-∞; 0]$ и $x = +\sqrt{y}$ при $x ∈ [0; +∞)$. На области определения $(-∞; +∞)$ обратной функции не существует, так как нарушается условие однозначности соответствия при $x = ±\sqrt{y}$.
### Теорема об обратной функции
**Пусть $y = f(x)$ определена, непрерывна и задана на промежутке $[a; b]$; $f(a) = \alpha, f(b) = \beta$. Функция $f(x)$ монотонно возрастает (убывает) на промежутке $[a; b]$. $[a; b] ⊂ D(f); [\alpha; \beta] ⊂ E(F)$. Тогда существует обратная функция, которая тоже будет являться непрерывной и будет монотонно возрастать (убывать).**

**В общем случае обратная функция наследует свойства исходной функции.**

Доказательство: Пусть функция $y = f(x)$ определена на $[a; b]$. Для определенности будем считать, что $y = f(x)$ монотонно возрастает.

В силу следствия из [[1 семестр БИЛЕТЫ 1-28#Следствие из теоремы Коши о промежуточном значении непрерывной функции|теоремы Коши о промежуточных значениях непрерывной функции]] отрезок $[a; b]$ единственным образом отображается на отрезок $[\alpha; \beta]$. Таким образом выполняется условие однозначности соответствия, и сохраняется свойство непрерывности для обратной функции.

Докажем, что $x = f^{-1}(y)$ тоже монотонно возрастает. Предположим противное:

$$
\forall y_1, y_2 ∈ [\alpha; \beta] : y_1 < y_2 ⇒ x_1 > x_2
$$

Получаем противоречие: при $x_1 > x_2$ и $f(x_1) < f(x_2)$ изначальная функция $y = f(x)$ монотонно убывает, а не возрастает. Следовательно, обратная функция должна монотонно возрастать, ч.т.д.
## Билет 23: Основные классы функций
### Основные классы элементарных функций
1) $y = x^a$ — степенные
2) $y = a^x$ — показательные
3) $y = \log_a x$ — логарифмические
4) $y = \sin x, y = \cos x, y = \mathop{\text{tg}} x, y = \mathop{\text{ctg}} x$ — тригонометрические
5) $y = \arcsin x, y = \arccos x, y = \mathop{\text{arctg}} x, y =  \mathop{\text{arcctg}} x$ — обратные тригонометрические
### Свойства степенных функций
Функция $y = x^n, n ∈ \mathbb{N}, x ∈ [0; +∞)$ непрерывна на данной области определения.

При $n = 1$ $f(x _n) = y _n = x _n, f(x _0) = y _0 = x _0 ⇒ \{f(x _n)\} \xrightarrow{n→∞}x _0, x _0 = f(x _0)$ — функция непрерывна.

При $n > 1$ $y = x^n$ непрерывна как произведение n непрерывных функций.

Функция $y = x^n, n ∈ \mathbb{N}, x ∈ [0; +∞)$ возрастает на данной области определения.

$$
\forall x _1, x _2 ∈ [0; +∞) : x _2 > x _1 ⇒ {x _2}^n \ge {x _1}^n
$$

$$
({x _2}^n - {x _1}^n) = (x _2 - x _1)({x _1}^{n-1} + {x _1}^{n-2}x _2 + {x _1}^{n-3}{x _2}^2 + \dots + {x _2}^{n-1})
$$

Оба множителя в правой части выражения положительные, поэтому ${x _2}^n - {x _1}^n > 0$ для всех $x ∈ [0; +∞)$.

При $x ∈ (-∞; 0]$ поведение функции зависит от значения $n$. При чётных значениях $n$ функция убывает в этой области, при нечётных — возрастает.

$$
\forall x _1, x _2 ∈ [0; +∞) : x _2 > x _1 ⇒ {x _2}^{2n} \le {x _1}^{2n}
$$

$$
({x _2}^{2n} - {x _1}^{2n}) = (x _2 - x _1)({x _1}^{2n-1} + {x _1}^{2n-2}x _2 + {x _1}^{2n-3}{x _2}^2 + \dots + {x _2}^{2n-1}) < 0
$$

(отрицательное число в нечётной степени является отрицательным, т.е. второй множитель является отрицательным)

или

$$
\forall x _1, x _2 ∈ [0; +∞) : x _2 > x _1 ⇒ {x _2}^{2n+1} \ge {x _1}^{2n+1}
$$

$$
({x _2}^{2n+1} - {x _1}^{2n+1}) = (x _2 - x _1)({x _1}^{2n} + {x _1}^{2n-1}x _2 + {x _1}^{2n-2}{x _2}^2 + \dots + {x _2}^{2n}) > 0
$$

(отрицательное число в чётной степени является положительным, т.е. второй множитель является положительным)

Рассмотрим функцию $y = x^p, p = \frac{m}{n}; m, n ∈ \mathbb{Z}$.

Функция $y = x^{\frac{1}{n}}$ непрерывна как обратная функции $y = x^n$. Соответственно, функция $y = x^{\frac{m}{n}}$ непрерывна как произведение нескольких непрерывных функций. Монотонность доказывается аналогично.

### Тригонометрические функции
Функции $y = \sin x$ и $y = \cos x$ являются непрерывными на всей числовой оси.

Докажем, что $\lim _{x→x _0} \sin x = x _0$.

$$
|\sin x-\sin x _0| = |2 \sin(\frac{x - x _0}{2}) \cos (\frac{x + x _0}{2})| = 2|\sin(\frac{x - x _0}{2})||\cos (\frac{x + x _0}{2})|
$$

Оба модуля всегда меньше или равны единицы, поэтому

$$
2|\sin(\frac{x - x _0}{2})||\cos (\frac{x + x _0}{2})| \le 2|\sin(\frac{x - x _0}{2})|
$$

$$
|\sin(x)| \le |x| ⇒ 2|\sin(\frac{x - x _0}{2})| \le 2 \frac{|x - x _0|}{2} = |x - x_0| ⇒ |\sin(x) - \sin(x _0)| \le |x - x _0|
$$

Запишем определение по Коши для того предела, который мы хотим доказать:

$$
\forall ε > 0 : \exists \delta = \delta(\varepsilon) : |x-x _0| < \delta, |\sin(x)-\sin(x _0)| < \varepsilon
$$

Получается, что если разность $|x - x _0|$ достаточно малая, то и $|\sin(x)-\sin(x _0)|$ тоже будет  достаточно мала, что доказывает непрерывность функции.

Непрерывность функции $y = \cos x$ доказывается аналогично.

## Билет 24: Первый и второй замечательные пределы
### Первый замечательный предел
$$
\lim_{x→0} \frac{sin(x)}{x}=1
$$

Доказательство:

$$
f(x) = \frac{\sin x}{x}; f(-x) = \frac{\sin(-x)}{-x} = \frac{-\sin x}{-x} = \frac{\sin x}{x} = f(x) ⇒f(x)\text{ — чётная функция}
$$

Следовательно, функция симметрична около оси $Oy$, и два односторонних предела не будут отличаться друг от друга, мы можем рассматривать любой из них.

Рассмотрим площади прямоугольного треугольника $OAB$, сектора $OAN$ и треугольника $OAN$ (см. график в десмосе: https://www.desmos.com/calculator/shmn5gwosi?lang=ru).

Очевидно, что $S_{\Delta OAN} < S_{\text{сектора }OAN} < S_{\Delta OAB}$. Выразим площади всех трёх фигур, используя $x$ для обозначения величины угла $\angle AON$, а $r$ для обозначения величины радиуса:

$$
\sin x = \frac{NH}{NO} ⇒ NH = NO · \sin x = r · \sin x
$$

$$
\mathop{\text{tg}} x = \frac{AB}{OA} ⇒ AB = OA · \mathop{\text{tg}} x = r · \mathop{\text{tg}} x
$$

$$
S_{\Delta OAN} = \frac{1}{2} · OA · NH = \frac{1}{2} · r · r · \sin x = \frac{1}{2} · r^2 · \sin x
$$

$$
S_{\text{сектора }OAN} = \frac{\pi · r^2 · x}{360°} = \frac{\pi · r^2 ·x}{2\pi} = \frac{1}{2} r^2 x
$$

$$
S_{\Delta OAB} = \frac{1}{2} · OA · AB = \frac{1}{2} · r · r · \mathop{\text{tg}} x = \frac{1}{2} r^2 \mathop{\text{tg}} x
$$

$$
S_{\Delta OAN} < S_{\text{сектора }OAN} < S_{\Delta OAB} ⇒ \frac{1}{2} r^2 \sin x < \frac{1}{2} r^2 x < \frac{1}{2} r^2 \mathop{\text{tg}} x ⇒ \sin x < x < \mathop{\text{tg}} x
$$

$$
x < \mathop{\text{tg}} x ⇒ x < \frac{\sin x}{\cos x} ⇒ \cos x < \frac{\sin x}{x};\quad \sin x < x ⇒ \frac{\sin x}{x} < 1
$$

Следовательно, получаем двойное неравенство:

$$
\cos x < \frac{\sin x}{x} < 1
$$

По [[1 семестр БИЛЕТЫ 1-28#Лемма о двух милиционерах|лемме о двух милиционерах]] имеем:

$$
\lim_{x→0} \cos x = 1, \lim_{x→0} 1 = 1 ⇒ \lim_{x→0} \frac{\sin x}{x} = 1
$$

, ч.т.д.
### Второй замечательный предел
$$
\lim_{x→∞} {(1+\frac{1}{x})}^x = e
$$

где $x$ — рациональное число.

Доказательство: 

Рассмотрим два случая: 1) $x → +∞$ и 2) $x → -∞$.

Возьмем число $x ∈ \mathbb{R}$, находящееся между натуральными числами $n$ и $n + 1$.

$$
\frac{1}{n + 1} < \frac{1}{x} < \frac{1}{n} ⇒ 1 + \frac{1}{n+1} < 1 + \frac{1}{x} < 1 + \frac{1}{n} ⇒ (1+\frac{1}{n+1})^n < (1+\frac{1}{x})^x < (1+\frac{1}{n})^{n+1} ⇒
$$

$$
⇒ \frac{(1+\frac{1}{n+1})^{n+1}}{1+\frac{1}{n+1}} < (1+\frac{1}{x})^x < (1+\frac{1}{n})^{n}(1+\frac{1}{n})
$$

$$
\lim_{n→∞} [\frac{(1+\frac{1}{n+1})^{n+1}}{1+\frac{1}{n+1}}] = \frac{e}{1} = e, \lim_{n→∞} [(1+\frac{1}{n})^{n}(1+\frac{1}{n})] = e · 1 = e

Следовательно, по лемме о двух милиционерах:

$$
\lim_{x→+∞} {(1+\frac{1}{x})}^x = e
$$

, ч.т.д.

Для второго случая введём переменную $y = -x$. Тогда

$$
\lim_{x→-∞} (1+\frac{1}{x})^x = \lim_{y→∞} (1-\frac{1}{y})^{-y} = \lim_{y→∞} (\frac{y-1}{y})^{-y} = \lim_{y→∞} (\frac{y}{y-1})^{y} = \lim_{y→∞} (\frac{y}{y-1}-1+1)^{y} =
$$

$$
= \lim_{y→∞} (\frac{y}{y-1}-\frac{y-1}{y-1}+1)^{y-1+1} = \lim_{y→∞} (\frac{1}{y-1}+1)^{y-1}(\frac{1}{y-1}+1) = e · 1 = e 
$$

, ч.т.д.

## Билет 25: Сравнение бесконечно малых и бесконечно больших величин
### "О" малое
Предположим, что $\alpha(x)$ и $\beta(x)$ одновременно являются бесконечно малыми или бесконечно большими величинами.

Если $\lim _{x→x _0} \frac{\alpha(x)}{\beta(x)} = q \ne 0, q = \text{const}$, тогда эти [[1 семестр БИЛЕТЫ 1-28#Определение функции|функции]] называют сравнимыми в окрестности точки $x _0$.

Если $\lim _{x→x _0} \frac{\alpha(x)}{\beta(x)} = 1$, тогда эти функции называют эквивалентными в окрестности точки $x _0$. Это утверждение можно записать как $\alpha(x) \sim \beta(x)$.

Пример: $\sin x \sim x, x → 0$.

Если $\lim _{x→x _0} \frac{\alpha(x)}{\beta(x)} = 0$, тогда:

1) если $\alpha(x)$ и $\beta(x)$ бесконечно малые, то $\alpha(x)$ будет иметь более высокий порядок бесконечной малости по сравнению с $\beta(x)$ в окрестности точки $x _0$. Также можно записать как $\alpha(x) = o(\beta(x))$.
2) если $\alpha(x)$ и $\beta(x)$ бесконечно большие, то $\beta(x)$ будет иметь более высокий порядок бесконечной величины по сравнению с $\alpha(x)$ в окрестности точки $x _0$.

Пример 1:

$$
\alpha(x) = x^2, \beta(x) = x, x → 0
$$

$$
x = 0.1, \alpha(x) = 0.01, \beta(x) = 0.1; x = 0.01, \alpha(x) = 0.0001, \beta(x) = 0.01
$$

$$
\lim_{x→0} \frac{\alpha(x)}{\beta(x)} = \lim_{x→0} \frac{x^2}{x} = 0 ⇒ x^2 = o(x) \text{ при }x → 0
$$

Пример 2:

$$
\alpha(x) = x^2, \beta(x) = x, x → +∞
$$

$$
\lim_{x→+∞} \frac{\beta(x)}{\alpha(x)} = \lim_{x→+∞} \frac{x}{x^2} = \lim_{x→+∞} \frac{1}{x} = 0 ⇒ x = o(x^2) \text{ при }x → +∞
$$
### "О" большое
Предположим, что отношение $\frac{\alpha(x)}{\beta(x)}$ ограничено в некоторой окрестности:

$$
|\frac{\alpha(x)}{\beta(x)}| \le const \text{ при }x = x _0
$$

Тогда говорят, что функция $\beta(x)$ доминирует над $\alpha(x)$ или $\alpha(x) = O(\beta(x))$.

Пример:

$$
\frac{\sin x}{x} < 1 ⇒ \sin x = O(x)
$$
## Билет 26: Классификация разрывов непрерывных функций
Пусть функция $y = f(x)$ [[1 семестр БИЛЕТЫ 1-28#Определение непрерывной функции|непрерывна]] в некоторой окрестности точки $x _0$, за исключением самой точки $x _0$, в которой функция терпит разрыв. Предположим, что для этой точки существуют [[1 семестр БИЛЕТЫ 1-28#Односторонние пределы|предел справа и предел слева]], которые равны друг другу. Тогда говорят, что в точке $x _0$ есть устранимый разрыв.

Пример:

Рассмотрим функцию $y = (sign(x))^2$.

$$
\lim _{x→0+0} (sign(x))^2 = 1; \lim _{x→0-0} (sign(x))^2 = 1
$$

Следовательно, $x = 0$ — устранимая точка разрыва.

Пусть существует предел справа, который равен конечному числу, и предел слева, который равен конечному числу, при этом предел справа не равен пределу слева. Тогда говорят, что в точке $x _0$ есть разрыв первого ряда.

Устранимая точка разрыва является частным случаем разрыва первого рода.

Пример:

Рассмотрим функцию $y = sign(x)$.

$$
\lim_{x→0+0} (sign(x)) = 1; \lim_{x→0-0} (sign(x)) = -1
$$

В точке $x = 0$ имеем разрыв первого рода.

Если хотя бы один из односторонних пределов не существует или не является конечным числом, то тогда говорят, что в рассматриваемой точке имеется разрыв второго рода.

Пример:

Рассмотрим функцию $y = \frac{1}{x}$.

$$
\lim_{x→0+0} \frac{1}{x} = +∞; \lim_{x→0-0} \frac{1}{x} = -∞;
$$

В точке $x = 0$ имеем разрыв второго рода.
## Билет 27: Основные теоремы о непрерывных функциях
### Критерий локального ограничения функции
**Функция, непрерывная на отрезке, является локально ограниченной на нём. Это означает, что функция является ограниченной в окрестности каждой точки этого отрезка.**

Доказательство: Используем определение функции, непрерывной в точке и запишем этот предел по определению Коши:

$$
\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : |x-x _0|<\delta, |f(x) - f(x _0)| < \varepsilon
$$

Для определённости полагаем $\varepsilon$ = $0.1$.

$$
|f(x)| < |f(x _0)| + 0.1, \forall x : |x - x _0| < \delta 
$$

, ч.т.д.
### Первая теорема Вейерштрасса
**Функция, непрерывная на отрезке, является ограниченной на этом отрезке.**

Доказательство:

Пусть функция $y = f(x)$ непрерывна на отрезке $[a; b]$:

$$
y = f(x) ∈ C[a; b]
$$

Для всех точек на отрезке $[a; b]$ существует интервал $I_0$, в котором функция будет локально ограничена. Обозначим множество таких интервалов как $\{I_0\}$. Таких интервалов может быть бесконечно много. В силу [[1 семестр БИЛЕТЫ 1-28#Лемма Бореля|леммы Бореля]] можно выделить конечное число интервалов. На каждом из этих интервалов функция ограничена, следовательно, она будет ограничена на всем отрезке $[a; b]$, ч.т.д.
### Вторая теорема Вейерштрасса
**Функция, непрерывна на отрезке, принимает на нём своё минимальное и максимальное значение.**

Доказательство:

Пусть функция $y = f(x)$ непрерывна на отрезке $[a; b]$. Докажем утверждение для минимального значения, для максимального доказательство аналогично.

Пусть $m$ — минимальное значение функции $f(x)$ на отрезке $[a; b]$. Запишем второе условие в определении инфимума:

$$
\forall \varepsilon > 0, ∃ x _n : m \le f(x _n) \le m + \varepsilon
$$

Предположим, что $x _n \xrightarrow{n→∞} x _0$. Тогда в силу непрерывности функции $f(x _n) \xrightarrow{n→∞} f(x _0)$.

Осуществляем [[1 семестр БИЛЕТЫ 1-28#Теоремы о предельном переходе в неравенствах|предельный переход в неравенствах]], используя [[1 семестр БИЛЕТЫ 1-28#Лемма о двух милиционерах|лемму о двух милиционерах]].

$m$ и $m + \varepsilon$ не зависят от $n$, а $f(x _n)$ сойдётся до значения $m$. Но $f(x _n)$ также сходится к $f(x _0)$. Следовательно, $f(x _0) = m$, ч.т.д.
## Билет 28: Условие Коши существования предела функции
### Условие Коши существования предела функции
**Для того, чтобы функция $y = f(x)$ имела предел, необходимо и достаточно, чтобы она удовлетворяла условию Коши:**

$$
\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : |x _2 - x _1| < \delta, |f(x _2) - f(x _1)| < \varepsilon
$$

Доказательство необходимого условия:

Пусть существует предел $\lim_{x→a} f(x) = b$ Покажем, что критерий Кошм выполняется. Используем определение предела и запишем его отдельно для $x _1, x _2$ из $\delta$-окрестности точки $a$:

$$
\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : |x _1 - a| < \frac{\delta}{2}, |f(x _1) - b| < \frac{\varepsilon}{2}
$$

$$
\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : |x _2 - a| < \frac{\delta}{2}, |f(x _2) - b| < \frac{\varepsilon}{2}
$$

Предположим, что точки $x _1$ и $x _2$ находятся по разные стороны от точки $a$. Тогда между ними будет максимальное расстояние $\delta$. Тогда

$$
|x _2 - x _1| < \delta, |f(x _2) - f(x _1)| < \varepsilon
$$

Поскольку точки $x _1, x _2, a$ были выбраны произвольно, это утверждения выполняется для всех точек отрезка, ч.т.д.

Доказательство достаточного условия:

Пусть выполняется критерий Коши. Покажем, что функция имеет предел.

Запишем критерий Коши:

$$
\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : |x _2 - x _1| < \delta, |f(x _2) - f(x _1)| < \varepsilon
$$

Нужно доказать следующее утверждение:

$$
\forall \{x _n\} \xrightarrow{n→∞}a ⇒ \{f(x _n)\} \xrightarrow{n→∞} b
$$

Рассмотрим последовательность $\{x _m\}$. которая также находится в $\delta$-окрестности точки $a$. Тогда $|x _m - x _n| < \delta$.

В силу критерия Коши $|f(x _m) - f(x _n)|  < \varepsilon$. Следовательно, выполняется критерий Коши для последовательности $\{x _m\}$, она является фундаментальной и имеет предел, ч.т.д.
