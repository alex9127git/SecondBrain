# Билет 29: Понятие равномерно непрерывной функции. Теорема Кантора о равномерной непрерывности
### Равномерно непрерывные функции
Функция $y = f(x)$ называется равномерно непрерывной на области $D$, если

$$
\forall \varepsilon > 0 : \exists \delta = \delta(\varepsilon) : |x _2 - x _1| < \delta, |f(x _2) - f(x _1)| < \varepsilon
$$

Замечание: для равномерно непрерывной функции значение $\delta$ может быть подобрано одним единственным образом для всех пар точек $x _1, x _2$. Для непрерывной фнкции значение $\delta$ будет меняться от точки к точке, то есть $\delta$ зависит не только от $\varepsilon$, но и от $x _0$.
### Теорема Кантора о равномерно непрерывной функции
**Функция, непрерывная на отрезке (замкнутом промежутке), является равномерно непрерывной на нём.**

Доказательство:

Пусть функция $y = f(x)$ непрерывна на отрезке $[a; b]$. Покажем, что на этом отрезке она является равномерно непрерывной.

Предположим противное:

$$
\exists \varepsilon_0 > 0 : \forall \delta = \delta(\varepsilon) : |x _2 - x _1| < \delta, |f(x _2) - f(x _1)| > \varepsilon_0
$$

Обозначим это условие звездочкой (\*).

Рассмотрим последовательность $\{x _n\} \xrightarrow{n→∞} x _0$. В силу непрерывности функции из определения Гейне следует, что $\{f(x _n)\} \xrightarrow{n→∞} f(x _0)$.

Рассмотрим последовательность $\{x _m\} \xrightarrow{m→∞} x _0$. В силу непрерывности функции из определения Гейне следует, что $\{f(x _m)\} \xrightarrow{m→∞} f(x _0)$.

Следовательно,

$$
\forall n > N(\varepsilon_0) : |x _n - x _0| < \delta, |f(x _n) - f(x _0)| < \varepsilon_0
$$

$$
\forall m > N(\varepsilon_0) : |x _m - x _0| < \delta, |f(x _m) - f(x _0)| < \varepsilon_0
$$

$$
\forall n, m > N(\varepsilon) : |x _n - x _m| < \delta, |f(x _n) - f(x _m)| < \varepsilon_0
$$

$$
\lim_{n→∞} (f(x _n) - f(x _m)) = 0
$$

, что противоречит условию (\*).
## Билет 30: Производная функции. Геометрическая интерпретация. Понятие касательной и секущей
Пусть [[1 семестр БИЛЕТЫ 1-28#Определение функции|функция]] $y = f(x)$ определена и задана в некоторой области $D$. Производные будем вычислять только во внутренних точках области. 
### Определение производной
Пусть точка $x _0$ — внутренняя точка области $D$. Пусть существует предел $\lim _{x→x _0} \frac{f(x) - f(x _0)}{x - x _0}$. Этот предел и называют производной функции $f(x)$ в точке $x _0$ и записывают как $f'(x _0)$ или $f' _{x _0}$.

Введём обозначения:

$\Delta x = x - x _0$ — приращение аргумента.

$\Delta f = f(x) - f(x _0)$ — приращение функции.

Таким образом, аргумент и значение функции можно переписать как $x = x _0 + \Delta x$ и $f(x) = f(x _0) + \Delta x$, а сам предел производной как $\lim _{\Delta x→0} \frac{\Delta f}{\Delta x}$.

Производная — предел отношения приращения функции к приращению аргумента.

Пример: $y = x^2$. Вычислить производную согласно определению.

$$
f'(x _0) = \lim_{\Delta x→0} \frac{f(x _0 + \Delta x) - f(x _0)}{\Delta x} = \lim_{\Delta x→0} \frac{(x _0 + \Delta x)^2 - (x _0)^2}{\Delta x} = 
$$

$$
\lim_{\Delta x→0} \frac{{x _0}^2 + 2x _0\Delta x + {\Delta x}^2 - {x _0}^2}{\Delta x} = \lim_{\Delta x→0} \frac{2x _0\Delta x + {\Delta x}^2}{\Delta x} = \lim_{\Delta x→0} 2x _0 + {\Delta x} = 2x _0
$$
### Уравнение касательной
Геометрическая интерпретация определения производной доступна по ссылке: https://www.desmos.com/calculator/qafmwuwomi?lang=ru

Запишем уравнение секущей, которая проходит через точки $A$ и $B$ с известными координатами:

$$
\frac{y - f(x _0)}{f(x _0 + \Delta x) - f(x _0)} = \frac{x - x _0}{x _0 + \Delta x - x _0}
$$

$$
y - f(x _0) = \frac{f(x _0 + \Delta x) - f(x _0)}{\Delta x} (x - x _0)
$$

$$
y = f(x _0) + \frac{f(x _0 + \Delta x) - f(x _0)}{\Delta x} (x - x _0)
$$

В пределе уравнение секущей превращается в уравнение касательной:

$$
y = f(x _0) + f'(x _0) (x - x _0) \text{ при } \Delta x → 0.
$$

Преобразуем уравнение касательной:

$$
y = f'(x _0) · x - f'(x _0) · x _0 + f(x _0)
$$

Получается уравнение прямой вида $y = kx + b$, где $k$ принимает значение $f'(x _0)$, а $b$ — $f'(x _0) · x _0 + f(x _0)$. Получаем, что тангенс угла наклона касательной равен производной функции в этой точке:

$$
k = \mathop{\text{tg}} \alpha = f'(x _0),
$$

где $\alpha$ — угол наклона между касательной и положительным направлением оси $Ox$.

Производная функции показывает её поведение на различных значениях $x$, что означает, что дифференцирование функции можно использовать для решения задач моделирования.
## Билет 31: Односторонняя производная
### Определение односторонней производной
Рассмотрим функцию $y = |x|$.

В точке $x = 0$ у функции $y = |x|$ невозможно провести касательную единственным образом. В этой точке нет производной, но можно вычислить так называемые односторонние производные:

$$
f'(x_0 + 0) = \lim_{\Delta x → 0; \Delta x > 0} \frac{\Delta f}{\Delta x}
$$

$$
f'(x_0 - 0) = \lim_{\Delta x → 0; \Delta x < 0} \frac{\Delta f}{\Delta x}
$$

Для функции $y = |x|$ имеем:

$$
f'(0 + 0) = \lim_{\Delta x → 0; \Delta x > 0} \frac{|x_0 + \Delta x| - |x_0|}{\Delta x} = \lim_{\Delta x → 0; \Delta x > 0} \frac{x_0 + \Delta x - x_0}{\Delta x} = \lim_{\Delta x → 0; \Delta x > 0} \frac{\Delta x}{\Delta x} = 1
$$

$$
f'(0 - 0) = \lim_{\Delta x → 0; \Delta x < 0} \frac{|x_0 + \Delta x| - |x_0|}{\Delta x} = \lim_{\Delta x → 0; \Delta x < 0} \frac{-x_0 - \Delta x + x_0}{\Delta x} = \lim_{\Delta x → 0; \Delta x > 0} \frac{-\Delta x}{\Delta x} = -1
$$

## Билет 33: Производная сложной функции. Производная обратной функции
### Производная сложной функции
**Пусть $y = f(x)$ дифференцируема (имеет [[1 семестр БИЛЕТЫ 29-35#Определение производной|производную]]) в точке $x _0$, $y _0 = f(x _0$), $z = g(y)$ дифференцируема в точке $y _0$, т.е. выполняются все условия существования [[1 семестр БИЛЕТЫ 1-28#Определение сложной функции|сложной функции]]. Тогда сложная функция $z = g(f(x))$ дифференцируема в точке $x _0$, и её производная может быть вычислена по формуле:**

$$
z'(x_0) = g'(y_0) · f'(x_0)
$$

Доказательство:

$$
z' = \lim_{\Delta x → 0} \frac{\Delta z}{\Delta x} = \lim_{\Delta x → 0} \frac{\Delta z · \Delta y}{\Delta x · \Delta y} = \lim_{\Delta x → 0} \frac{\Delta z}{\Delta y} · \lim_{\Delta x → 0} \frac{\Delta y}{\Delta x}
$$

Рассмотрим производную геометрически: https://www.desmos.com/calculator/qafmwuwomi?lang=ru

При $\Delta x → 0$ приращение функции $\Delta f$ тоже будет стремиться к нулю. Следовательно, верно следующее преобразование:

$$
\lim_{\Delta x → 0} \frac{\Delta z}{\Delta y} · \lim_{\Delta x → 0} \frac{\Delta y}{\Delta x} = \lim_{\Delta y → 0} \frac{\Delta z}{\Delta y} · \lim_{\Delta x → 0} \frac{\Delta y}{\Delta x} = z'_y · y'_x
$$

### Производная обратной функции
**Пусть функция $y = f(x)$ дифференцируема (имеет [[1 семестр БИЛЕТЫ 29-35#Определение производной|производную]]) в точке $x _0$. Пусть существует обратная функция $x = f^{-1}(y)$, причём производная $f'(x _0) \ne 0$. Тогда производная обратной функции может быть вычислена по формуле:**

$$
(f^{-1})'_{x_0} = \frac{1}{f'(x_0)}
$$

Доказательство:

$$
(f^{-1})' = \lim_{\Delta y → 0} \frac{\Delta x}{\Delta y} = \lim_{\Delta y → 0} \frac{1}{\frac{\Delta y}{\Delta x}} = \frac{1}{\lim_{\Delta x → 0}{\frac{\Delta y}{\Delta x}}} = \frac{1}{f'(x _0)}
$$

, ч.т.д.
