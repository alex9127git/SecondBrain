<u>Опр</u>
Под **множеством** будем понимать некоторую совокупность различных между собой объектов, мыслимых как единое целое.
<u>Пр</u> 
A = {1; 2; 3; 4; 5}

**Множество задано, если относительного любого элемента внешнего мира, можно сказать является ли оно элементом этого множества.**

Два множества **равны** (А = В) тогда и только тогда, когда они состоят из одних и тех же элементов.
1) Если x ∈ А, то х ∈ В
2) Если х ∈ В, то х ∈ А
<u>Пр</u>
A = {1; 2}, B = { {1;2} }
A **≠** B
B = { A }

<u>Пр</u>
А - множество всех **чётных натуральных чисел**.
В - множество натуральных чисел, которые представлены в виде **суммы двух нечётных чисел**.
Доказать, что А = В.

Доказательство:
А = { x ∈ N :- x = 2n, n ∈ N }
B = { x ∈ N :- x = p + q, p ≡ 1 (mod 2), q ≡ 1 (mod 2), p, q ∈ Z }
Докажем, что
1) ∀x ∈ A => x ∈ B
2) ∀x ∈ B => x ∈ A

Докажем 1)
Пусть х - произвольное число из А, х = 2n, n ∈ N
x = 2n + 1 - 1
x = (2n - 1) + 1
(2n - 1) ≡ 1 (mod 2)
1 ≡ 1 (mod 2),
Тогда очевидно, что х представлен в виде суммы двух нечётных натуральных чисел
=> x ∈ B

Докажем 2)
Пусть х - произвольный элемент из В
x = (2p - 1) + (2q - 1), p, q ∈ N
x = 2p + 2q - 2 = 2(p + q - 1) => x ∈ A
чтд

<u>Опр</u>
Если множество конечное, т.е. содержит конечное число элементов, тогда его называют **конечным**, в противном случае **бесконечным**.
<u>Опр</u>
**Пустым** множеством называется множество не содержащее в себе ни одного элемента ∅ (Мощность такого множества = 0)

### Мощность конечного множества

**|A| — количество элементов этого множества** 
A = {○, △, ▯}, |A| = 3

<u>Опр</u>
Если ∀x ∈ A : x ∈ B, то говорят, что множество А является **подмножеством** множества В
(A ⊆ B)
// ⊆ ~ <= 
**Пустое множество является подмножеством любого множества**
∅ ⊆ A, A - любое множество

<u>Опр</u>
A называют **собственным подмножеством** В, если А является подмножеством В, но не равно ему
A ⊂ B
ИЛИ
A ⊆ B, НО A **≠** B 

<u>Пр</u>
A = { 1;  2;  { 1 }; 3 }
1 ∈ A
2 ∈ A
{1} ∈ A ( из-за элемента с индексом 2 )
{1} ⊆ A ( из-за элемента с индексом 0 )
{ {1} } ⊆ A ( из-за элемента с индексом 2 )
∅ ⊆ A

**Свойство отношения включений**
1) A ⊆ A (рефлексивность)
2) A ⊆ B и B ⊆ C => A ⊆ C (транзитивность)
3) A ⊆ B и B ⊆ A => A = B (антисимметричность)

### Булеан множества(показательное множество) P(A) или $2^A$
<u>Опр</u>
Булеан множества - множество всех подмножеств множества А  

**Мощность булеана** 
$|2^A|$ = $2^{|A|}$ 
<u>Пр</u>
A = { 1; 2; 3 }, |A| = 3
$2^{A}$ = { ∅, {1}, {2}, {3}, {1; 2}, {1;3}, {2; 3}, {1; 2; 3} }
$|2^A|$ = 8

**Способы задания множеств**
1) Если множество А - конечное, то его элементы можно задать перечислением                      А = {$a1; a2; a3; a4; ... ; an$}, |A| = n
2) Если множество А - бесконечное, тогда можно
	2.1) задать множество используя рекурсию :
	Если n - натуральное, то (n + 1) - тоже натуральное
	
	2.2) Задать множество с помощью свойств его элементов:
	Пусть P(x) - утверждение, заключающееся в том, что х обладает свойством P, тогда
	A = { x : P(x) } или B = { x ∈ K : F(x) }

<u>Пр</u>
А = { x ∈ R :- 0 ≤ x ≤ 1 } - бесконечное множество, содержащее любые х в промежутке [0; 1]
А = { x :- x ≡ 0 (mod 2), x ∈ N } - конечное множество, содержащее чётные натуральные числа

**Парадокс Рассела**
Пусть M - множество всех множеств, тогда очевидно, что M ⊆ M, т.е. ∃ множества содержащие себя, как свой элемент (1)

Пусть Х - множество, которое не содержит себя, как свой элемент
Пусть Y - множество всех таких множеств Х (2)
Т.е. Y - множество всех множеств, не содержащих себя, как свой элемент (3)

Содержит ли множество Y себя, как свой элемент ?
1) Пусть содержит, т.е. Y ∈ Y. Получили противоречие с (3), т.к. множество содержит только такие множества, которые не содержат себя, как свой элемент
2) Пусть НЕ содержит, т.е. Y ∉ Y. Получили противоречие с (2), т.к. множество должно содержать само себя
