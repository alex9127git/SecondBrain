[[Лекция 3.]]

Пусть Х и Y - некоторые множества.
<u>Опр</u>
Декартово (прямое) произведение Х **×** Y - множество всех пар (х, у).
Х **×** Y = { (x, y) :- x ∈ Х, y ∈ Y}

<u>Пр</u>
A = {1; 2; 3}
B = {" + "; " - "}
![[ТаблицаКомбинаций.png]]
B **×** A = { $(1; +), (1; -), (2; +), ..., (3; +), (3; -)$ }

**СВОЙСТВО** A **×** B
Мощность декартова произведения $|A × B| = |A| * |B|$

<u>Опр</u> Декартово произведение n множеств:
$x_{1} × x_{2} × x_{3} × ... × x_{n}$ = { ($x_{1} ; x_{2} ; x_{3} ; ... ; x_{n}) :-x_{1} ∈ X_{1}, x_{2} ∈ X_{2}, ..., x_{n} ∈ X_{n}$ } - Набор n элементов, кортеж длины n.

**Замечание!**
 $X × Х × Х × ... × Х$ } = $Х^n$ - декартова степень множества Х.

Свойства:
1) (A or B) × C = (A × C) or (B × C)
2) (A & B) × C = (A × C) & (B × C)
3) (A \ B) × C = (A & NOT(B)) × C = (A × C) & (NOT(B) × C) = (A × C) \ (В × С)

Доказательство 2 свойства:
1) Если (х; y) ∈ (A & B) × C, тогда (х; y) ∈ (A × C) & (B × C)
2) Если (х; y) ∈ (A × C) & (B × C), тогда (х; y) ∈ (A & B) × C
1)Пусть д.п. (х; y) ∈ (A & B) × C => по опр д.п. x ∈ (A & B), y ∈ C =>
=> x ∈ A, y ∈ C и x ∈ B, y ∈ C => (x, y) ∈ A × C и (x, y) ∈ B × C
Т.е. принадлежит <u>пересечению</u>.
2)Пусть д.п. (х; y) ∈ (A × C) & (B × C). 
=> по опр д.п. x ∈ A, y ∈ C & x ∈ B, y ∈ C => y ∈ C, x ∈ A & B =>
=> (x, y) ∈ (A & B) × C

### Отображение. Типы отображений.
Х и Y - некоторые множества.
<u>Опр</u> 
Отображeнием f : Х -> Y называется называется тройка <x, y, f>, где Х и Y - не пустые множества, а f - правило (закон), которое каждому х из Х ставит в соответствие y из Y.

<u>Пр</u>
Х = Y = R, y = $x^{3}$
<u>Пр</u>
Х = Y = {1; 2; 3}
f : X -> Y
f(1) = 1
f(2) = 1
f(3) = 2

<u>Опр</u>
Отображение f : X -> Y называется **сюръективным**, если:
∀ y ∈ Y ∃ x ∈ X  :- f(x) = y

Пояснение детальнее:
**Сюръективное отображение** это отображение между двумя множествами, при котором каждый элемент второго 
множества является образом хотя бы одного элемента первого множества. Это означает, что 
отображение "покрывает" все элементы второго множества своими значениями. 
Например, функция f(x)=2x+1 является сюръективным отображением множества 
действительных чисел на себя, так как для любого вещественного числа y существует такое  вещественное число x, такое что f(x)=yf(x)=y

<u>Опр</u>
Отображение f : X -> Y называется **инъективным**, если
$f(x_{1}) = f(x_{2})$ => $x_1 = x_2$
Это означает, что разные элементы из множества X имеют разные образы в множестве Y.
f(x) - образ, x - прообраз.
Иначе говоря, из равенства образов следует равенство прообразов.

<u>Опр</u>
Отображение f : X -> Y называется **биективным**(взаимно однозначным), если оно **сюръективно** и **инъективно**.

<u>Пр</u>
$f(x) = x^2$,  $R^{+}$ = { x ∈ R :- x >= 0 }
1) 
$f_{1}(x) = x^2$ ; $f_1 : R -> R^+$
$∀ y ∈ R^+  ∃ x ∈ R  :-\ x = ±\sqrt{y}$
=> $f_1(x)$ - **сюръективно**
2) 
$f_2(x) = x^2$, $f_2 : R^+ -> R$
$∀ y ∈ R \\\  ∃\ x ∈ R^+$ - неверно. Т.к. для y < 0 не существует f(x) !!!
$x^2 = y < 0$
=> $f_2$ - не является сюръективным
3) 
Пусть $f(x_1) = f(x_2)$
$\sqrt{{x_{1}}^2} =  {\sqrt{x_{2}}^2}$
${\sqrt{x_{1}^2}} =  \sqrt{{x_{2}}^2}$
$|{x_{1}}| =  |{x_{2}}|$
$x_1 = x_2$ т.к. $x_1, x_2 ∈ R^+$
=> $f_2$ - инъективное 

$f_3$ = $x^2$, $f_3 : \ R^+ -> R^+$ - биективное отображение
### Обратные отображения
<u>Опр</u>
Отображение f : X -> X, тогда **f - преобразование множества Х**.

<u>Опр</u>
Тождественным преобразование множества Х называется преобразование:
$e_x :-\ e_{x}(x) = x, ∀ x ∈ X$

<u>Пр</u>
1) Х = N
$f : N -> N, f(x) = 1 * x$ - тождественное преобразование.

2) Х = { множество матриц n x n }
$$
E_n = \begin{pmatrix}
    1 & 0 & \cdots & 0 \\
    0 & 1 & \cdots & 0 \\
    \vdots & \vdots & \ddots & \vdots \\
    0 & 0 & \cdots & 1
\end{pmatrix}
$$
 Тогда $f : X -> X$
 $f(X) = E * X$ - тождественное преобразование
 
 ∀  ∈  ∃
<u>Опр</u>
Пусть $f : X -> Y,\ g : Y -> Z$ - некоторые отображения
Суперпозиция этих отображений:
$gf : X -> Z :-\ (gf)(x) = g(f(x)), ∀ x ∈ X$

<u>Пр</u>
$f(x) = e^x, g(y) = sin y$
=> $(gf)(x) = g(f(x)) = sin(e^x)$

<u>Опр</u>
Пусть f : X -> Y и g : Y -> X и g : X -> Y, тогда отображение g называется обратным отображением к f (отображение f называется отображением к g), если выполнены 2 условия:
$$
\begin{cases}
    f(g(y)) = y \\
    g(f(x)) = x
\end{cases}
$$

<u>Th</u>
Отображение f : X -> Y имеет обратное отображение тогда и только тогда, когда f является **биективным** отображением.

Доказательство:
Н: Если f имеет обратное отображение => f - биективное(инъективное И сюръективное)
Пусть f имеет обратное отображение => ∃ g : Y -> X :- 
$$
\begin{cases}
    f(g(y)) = y \\
    g(f(x)) = x
\end{cases}
$$
Покажем, что f - сюръективное отображение:
$∀ y ∈ Y \\\  ∃\ x ∈ X :- \ f(x) = y$ - ?
Возьмём  произвольный y ∈ Y и найдём х = g(y), x ∈ X
Тогда f(x) = f(g(y)) = y, из первого условия в слу => f - сюръективное отображение.

Покажем, что f - инъективное отображение(из равенства образов следует равенство прообразов)
Пусть $f(x_1) = f(x_2), x_1, x_2  ∈ X$, тогда 
$x_1 = g(f(x_1)) = g(f(x_2)) = x_2 => x_2 = x_1$
=> f - инъективное отображение

Тогда f - биективное отображение(т.к. и инъективное и сюръективное)

Д: 
f - биективное отображение. Докажем, что существует обратное отображение.
Из условия f - инъективное и сюръективное отображение.
Определим g : Y -> X, ∀ y ∈ Y   ∃ x ∈ X :- g(y) = x, f(x) = y
$$
\begin{array}{c}
    \forall x \in X \\
    \forall y \in Y
\end{array}
\begin{cases}
    f(g(y)) = f(x) = y \\
    g(f(x)) = g(y) = x
\end{cases}
$$
=> g - обратное отображение f

<u>Th</u>
Если обратное отображение существует, то оно единственное.

<u>Пр</u>
f : R -> R, $f(x) = \frac{2x - 5}{3}$
Задача: построить обратное отображение, если это возможно.
Решение:
Докажет, что f - биективное отображение:
Ин: $f(x_1) = f(x_2) => x_1 = x_2,\ x_1, x_2 ∈ R$
$\frac{2x_1 - 5}{3}$ = $\frac{2x_2 - 5}{3}$ => $\ x_2 = x_1$
Сюр: ∀ y ∈ R   ∃ x ∈ R :- f(x) = y
=> $x = \frac{3y + 5}2$
f(x) = f($\frac{3y + 5}2$) = y
$f^-1$ = $\frac{3x + 5}2$
Т.е. можно построить обратное отображене
